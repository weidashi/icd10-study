📋 一、附加编码概述——为什么一个诊断需要编多个码？

1.1 主编码 vs 附加编码的定义

在 ICD-10 的编码体系中，"一个诊断一个码"是最基本的情况，但 大约30%-40% 的住院病案需要编多个码才能完整描述一个疾病或事件。这些"辅助性"的编码统称为附加编码（Supplementary Codes）。

核心定义：附加编码是对主编码的补充说明，提供主编码无法表达的额外信息维度——例如疾病的组织学类型（形态学）、外部原因（损伤机制）、临床表现（星号码）等。

1.2 附加编码的四大类型——一张表说清楚

类型码位范围何时触发一句话规则
形态学编码（M码） M8000/0 ~ M9989/3 凡使用 C00-D48（肿瘤章）的任何编码 肿瘤编码的"必选项"——C/D 码必须配 M 码，否则病案首页不完整
外因编码 V01-Y98 凡使用 S00-T98（损伤中毒章）编码时 说明"怎么伤的"——永远附加，绝不能做主诊断
星剑号双重分类 散在于各章（带 † 和 * 标记） 疾病同时涉及病因+突出临床表现 剑号 (†) 主码 + 星号 (*) 附加码，两个都必须填
其他附加信息 如 Z 章、R 章转换 放化疗史（Z51）、治疗后状态（Z48）、器官切除后（Z90）等 提供"背景信息"，影响 DRG 分组的并发症/合并症计算
⚠️ 附加编码的核心铁律：附加编码永远不能单独做主诊断——它们没有独立的疾病意义，只是对主诊断的"注解"。如果把外因码 V01-Y98 填在主诊断位置，质控直接判错，DRG 无法入组。

1.3 什么时候需要多码？决策树

1

这个诊断码是 C 或 D 开头吗？

是 → 必须加形态学 M 码（附加） | 否 → 继续判断

2

这个诊断码是 S 或 T 开头吗？

是 → 必须查外因，编 V/W/X/Y 码（附加） | 否 → 继续判断

3

是 → 必须同时编对应的星号/剑号码 | 否 → 单码即可

4

是 → 考虑加 Z 章编码（附加） | 否 → 编码完成

🔬 二、形态学编码（M编码）——肿瘤的"组织学身份证"

2.1 M 编码的结构拆解

M 编码（Morphology Code）源自 ICD-O（国际肿瘤疾病分类），一套完整的 M 编码由两部分组成：

M 8 1 4 0  /  3
│ └─┬──┘    │  │
│   四位数字   │  动态码（肿瘤行为）
│  表示组织学  │
│  类型       │
首字母M      分隔符
(Morphology的
 缩写)

四位数字（M8000 - M9989）：表示肿瘤的组织学类型——即显微镜下看到的是什么细胞。例如 M8140 = 腺癌，M8070 = 鳞状细胞癌，M8720 = 恶性黑色素瘤。

动态码（/后一位数字）：表示肿瘤的生物学行为——良性还是恶性？原发还是转移？这是区分"C码"（恶性）还是"D码"（良性/原位）的决定性依据。

2.2 六种动态码——肿瘤行为的"红绿灯"

动态码含义对应的 ICD-10 部位码范围临床含义标签
/0良性D10-D36不会转移，生长慢，边界清楚 良性
/1良性或恶性未肯定（交界性/动态未定）D37-D48病理报告写了"不能确定良恶性"，需随访 未肯定
/2原位癌（上皮内瘤变Ⅲ级）D00-D09癌细胞限于上皮层内，未突破基底膜——可治愈 原位
/3恶性，原发部位C00-C76, C80-C97有侵袭和转移能力，"癌症"的最常用动态码 恶性原发
/6恶性，转移部位（继发恶性肿瘤）C77-C79从其他部位转移过来的——编码时部位选转移所在器官 转移
/9恶性，原发或转移未肯定C80（未特指部位恶性肿瘤）病理明确是恶性，但分不清原发还是转移 恶性未定
⚠️ 最常见的形态学报错：良恶性不匹配！
例如：部位编码填了 C34.9（肺恶性肿瘤），但形态学写了 M8140/0（良性腺瘤）→ 质控系统直接报错，动态码必须与部位码的性质（C=恶性 / D=良性或原位）一致。

2.3 肿瘤编码的黄金法则："先形态学，后部位"

很多新手习惯先看"肿瘤长在哪"再查编码，但 ICD-10 的肿瘤编码有严格的先后顺序：

❌ 错误顺序

1. 先查部位（"肺肿瘤"）→ 翻到 C34.x

2. 再查形态学 → 发现病理是"小细胞癌"

3. 强行把 M8041/3 配到 C34.9

问题：跳过了索引中的重要指引信息，可能错过特定的合并条目

✅ 正确顺序

1. 查形态学主导词 → "癌→小细胞 (M8041/3)"

2. 看索引指引 → "另见 肿瘤，恶性"

3. 查肿瘤表→按部位定位 C34.9

4. 返回第一卷核对"不包括"注释

优势：形态学编码决定了肿瘤表查哪个栏（良性栏/恶性栏/原位栏），确保不会查错栏

📌 肿瘤编码四步法（标准流程）：

1

根据病理报告确定形态学编码

主导词一般为"癌""腺癌""肉瘤""黑色素瘤"等 → 查到 Mxxxx/x

关键：以术后石蜡病理为主要依据，术中冰冻病理准确率约70%，仅作参考

2

根据形态学编码的动态码，选择肿瘤表的正确栏

/0 → 良性栏 | /1 → 动态未定栏 | /2 → 原位栏 | /3 → 恶性原发栏 | /6 → 继发栏

3

在对应栏中按解剖部位查找部位编码

"肿瘤表"中按器官/组织名称查 → 得到 C 或 D 的部位码

4

返回第一卷核对"包括""不包括"注释

确认部位编码与形态学编码在分类体系中无冲突

💣 三、形态学编码的高频陷阱——避坑指南

陷阱1：M8140/3 不是万能"腺癌"码

M8140/3 是"腺癌，NOS（未特指）"的通用编码，但某些特殊部位的腺癌有专属形态学编码，不能套用 M8140/3：

临床诊断❌ 错误 M 码✅ 正确 M 码解释
子宫内膜样腺癌M8140/3M8380/3子宫内膜样腺癌是 I 型子宫内膜癌的特定亚型，ICD-O 给了独立编码
甲状腺乳头状癌M8140/3M8260/3乳头状癌是甲状腺癌的最常见类型，有专属编码
胃印戒细胞癌M8140/3M8490/3印戒细胞癌是弥漫型胃癌的特殊组织学类型
肝细胞癌M8140/3M8170/3肝细胞癌不是"腺癌"，是独立的形态学类型
💡 避坑原则：病理报告写了具体的组织学类型（如"乳头状""印戒细胞""透明细胞""黏液性"）→ 一定要在索引中查这个具体类型，而不是默认填 M8140/3。

陷阱2：原位癌（/2）与恶性（/3）的界限——异型增生怎么编？

消化道病理报告中常见"异型增生（Dysplasia）"和"上皮内瘤变（Intraepithelial Neoplasia, IEN）"这两个术语。它们与编码的对应关系如下：

病理术语对应动态码部位编码说明
轻度异型增生 / 低级别上皮内瘤变不够原位癌标准不能编 D00-D09不属于原位癌，通常归入良性或炎症范畴
重度异型增生 / 高级别上皮内瘤变 M8010/2D00-D09（原位癌）等同于原位癌——这是中国病理界的共识
黏膜内癌（浸润限于黏膜层）M8010/3C 码（恶性）已突破基底膜但限于黏膜层，按恶性编码
⚠️ 关键记忆点：病理写"重度异型增生"或"高级别上皮内瘤变"→ 部位码编 D 开头（原位癌），形态学 M8010/2。这个规则在消化道肿瘤（食管、胃、结肠）中特别常见，编码员必须掌握。

陷阱3：没有病理报告时，M 码怎么填？

某些情况下患者未做病理检查（如晚期患者仅做姑息治疗、放化疗患者），但 ICD-10 规则要求 C00-D48 类目必须填写形态学编码。此时：

❌ 错误做法

因为没做病理检查，所以不填 M 码

后果：病案首页不完整，质控扣分

✅ 正确做法

使用通用形态学编码：

M8000/3 恶性肿瘤（无法获取组织学类型时）

M8010/3 癌，NOS（临床可判断为癌但无病理分型）

M8000/1 动态未定肿瘤（良恶性无法判断时）

🚑 四、外因编码（第二十章 V01-Y98）——损伤中毒的"来历说明"

4.1 外因编码的核心规则

ICD-10 第十九章（S00-T98）描述的是损伤本身（"什么伤"），而第二十章（V01-Y98）描述的是损伤的外在原因（"怎么伤的"）。二者是强制配对关系：

主诊断：S06.9 颅内损伤（损伤本身）
附加码：V03.1 行人被小汽车碰撞（外因）

主诊断：T58 一氧化碳中毒（中毒本身）
附加码：X47.- 气体和蒸气的意外中毒（外因）
⚠️ 外因编码三大铁律：
  1. 永远附加——外因码绝不能填在"主要诊断"栏，只能填在"其他诊断"或"外因"栏
  2. 有损伤就有外因——只要用了 S-T 码，就必须查外因编码
  3. 意图是关键——意外（Accident）、故意自害（Suicide）、加害（Assault）、意图不确定，编码完全不同

4.2 外因编码的分类结构

码位范围类别典型场景
V01-V99运输事故行人被汽车撞（V03）、骑自行车摔伤（V18）、汽车乘员事故（V43）
W00-W99跌倒、坠落地面滑倒（W01）、楼梯跌倒（W10）、床上跌落（W06）
X00-X09暴露于火焰/热火灾烧伤、烫伤
X40-X49有毒物质的意外中毒煤气中毒（X47）、农药中毒（X48）、药物中毒（X41-X44）
X60-X84故意自害自缢、服毒自杀、跳楼——编码与"意外"完全不同
X85-Y09加害（他伤）被他人殴打、刺伤、枪伤
Y10-Y34意图不确定的事件无法判断是意外还是自害/加害

4.3 外因编码真实案例解析

案例A：煤气中毒

临床场景：患者在家中因煤气泄漏中毒昏迷

主诊断：T58 一氧化碳中毒

外因码：X47.- 气体和蒸气的意外中毒及暴露

索引路径：意外中毒→煤气→X47

注意：如果是故意自害= X67；意图不确定= Y17

案例B：跌倒致股骨颈骨折

临床场景：80岁患者家中平地行走跌倒

主诊断：S72.0 股骨颈骨折

外因码：W01.- 在同一平面上滑倒、绊倒

索引路径：跌倒→平地滑倒→W01

注意：床上跌落= W06，楼梯跌倒= W10

案例C：手术材料并发症

临床场景：腹股沟疝修补术后，补片引起疼痛需取出

主诊断：T85.6 不可吸收性手术材料引起的机械性并发症

外因码：Y83.2 吻合、分流或移植手术作为异常反应的原因

注意：这是"手术后并发症"的特殊外因类型

⭐ 五、星剑号系统——病因+表现的双码模式

这部分在 Day 5 已详细讲解，此处仅做快速回顾——从"附加编码"的角度理解星剑号：

角色标记编码位置驱动 DRG？
剑号 (†)：病因码，主诊断 † / Dagger 主要诊断栏 ✅ 是
星号 (*)：临床表现码，附加诊断 * / Asterisk 其他诊断栏 ❌ 否（仅作信息补充）
📌 星剑号 vs M 编码的区别：
- 星剑号 = 一种编码体系（病因+表现两个诊断码），两个码都是 ICD-10 标准码
- M 编码 = 一种编码格式（Mxxxx/x），用于描述组织学特征，是肿瘤码 C/D 的"属性"

🔬 六、重难点深度解析：形态学编码决定了你查肿瘤表的哪个"栏"

这个知识点看似简单，却是编码考试和日常工作中最容易翻车的地方。让我们用一个完整案例走通全流程。

实战案例：结肠息肉镜下有"重度异型增生"

场景：患者行肠镜检查，发现乙状结肠一2cm息肉，予以镜下切除。术后病理回报："管状腺瘤伴重度异型增生（高级别上皮内瘤变），切缘阴性"。

问题：这个息肉的诊断编码应该是什么？

编码全流程推导：

1

确定形态学编码——病理报告说了什么？

"重度异型增生" = "高级别上皮内瘤变" = 等同于原位癌

索引：癌 → 上皮内 (M8010/2) → 另见 肿瘤，原位

✅ 形态学编码：M8010/2（原位癌）

2

选择肿瘤表的"正确栏"——/2 决定了查哪个栏

M8010//2 → 动态码为原位癌 → 必须查肿瘤表的"原位"栏

易错点：如果不先确定形态学就直接查"结肠→恶性"，你会翻到 C18.x（结肠恶性肿瘤），这就错了！

3

在"原位"栏中查找乙状结肠

肿瘤表原位栏 → 结肠 → 乙状结肠 → D01.0

✅ 部位编码：D01.0（结肠原位癌，乙状结肠）

4

核对卷一 → 形态学+部位的一致性

D01.0（结肠原位癌）+ M8010/2（原位癌） → 良恶性一致 ✅

完整编码：D01.0（主诊断）+ M8010/2（形态学附加码）

⚠️ 如果跳过"先形态学"这一步会怎样？

编码员若先查部位"乙状结肠肿瘤 → C18.7（乙状结肠恶性肿瘤）"，然后配上病理"重度异型增生 → M8010/2（原位癌）"，结果：C18.7（恶性）+ M8010/2（原位）= 良恶性不匹配 —— 质控直接报错！

这就是为什么"先形态学、后部位"不是建议，是铁律。

✅ 答案：

M8140/3：恶性原发（腺癌）→ 部位码在 C 章（C00-C76）

M8000/0：良性肿瘤 → 部位码在 D 章（D10-D36）

M8010/2：原位癌 → 部位码在 D 章（D00-D09）

M8000/6：恶性肿瘤转移 → 部位码在 C 章（C77-C79，继发恶性肿瘤）

M8000/1：动态未定（良恶性未肯定）→ 部位码在 D 章（D37-D48）

💡 记忆口诀：C 章 = /3 或 /6； D 章 = /0、/1 或 /2。"恶C良D，三六C来零一二D"。

基础题 · 形态学编码应用 PRACTICE

某患者出院诊断"左肺上叶鳞状细胞癌"，术后病理明确为鳞状细胞癌，未提及具体亚型。请给出完整的诊断编码（含形态学）。

✅ 完整编码推导：

第一步 · 形态学：查索引 → 癌 → 鳞状细胞 (M8070/3) → 另见 肿瘤，恶性

第二步 · 部位：肿瘤表"恶性原发"栏 → 肺 → 上叶 → C34.1

第三步 · 核对卷一：C34.1 肺上叶恶性肿瘤，不包括 C33（气管恶性肿瘤）——无误

✅ 最终编码：主诊断 C34.1（左肺上叶恶性肿瘤）+ 形态学 M8070/3（鳞状细胞癌）

基础题 · 外因编码规则 PRACTICE

判断以下说法是否正确，错误的请修正：

  1. "外因编码 V01-Y98 可以作为主要诊断编码"
  2. "如果患者在家中被热水烫伤，只需要编 T22.1（上肢烧伤），不需要外因码"
  3. "煤气中毒只需要编 T58，不需要编外因码"
点击查看答案

✅ 答案：

1. ❌ 错误。外因编码 V01-Y98 绝对不能做主诊断。它们是附加编码，只能填在"其他诊断"或"外因"栏。填在主诊断位置 → 质控直接判错。

2. ❌ 错误。只要主诊断用了 S-T 码（包括 T22.1），就必须附加外因码。烫伤 → 应加 X11.-（接触热自来水）或 X12.-（接触其他热液体）。

3. ❌ 错误。煤气中毒 = T58（中毒本身） + 外因码。如果是家中意外 → X47.-（气体意外中毒）；如果是故意自害 → X67.-。不加外因码 → 损伤编码不完整。

进阶题 · 形态学陷阱 CHALLENGE

某患者因子宫内膜癌入院手术，术后病理报告："子宫内膜样腺癌，FIGO Ⅰ级，浸润深度＜1/2肌层"。

编码员小王查"腺癌 → M8140/3"，然后查部位"子宫 → 内膜 C54.1"，给出编码：C54.1 + M8140/3。

问题：这个编码对吗？如果不对，错在哪里？正确编码应该是什么？

点击查看答案

❌ 小王的编码有误！

错误原因："子宫内膜样腺癌"不是普通的"腺癌"——它有专属的形态学编码：M8380/3（子宫内膜样腺癌），而不是 M8140/3（腺癌，NOS）。这是编码审核中排名靠前的高频错误。

✅ 正确推导：

查索引：腺癌 → 子宫内膜样 (M8380/3) → 特指部位 → 见 肿瘤，恶性

肿瘤表恶性栏：肿瘤 → 子宫 → 内膜 → C54.1

✅ 正确编码：C54.1（子宫内膜恶性肿瘤）+ M8380/3（子宫内膜样腺癌）

💡 教训：病理报告写了"子宫内膜样""乳头状""印戒细胞""透明细胞"等具体亚型时，一定要在索引中查这个具体亚型，不能偷懒默认用 M8140/3。

进阶题 · 原位 vs 恶性判断 CHALLENGE

下面是三个不同的病理报告场景，请分别给出正确的部位编码 + 形态学编码：

场景部位病理报告
A胃窦"管状腺癌，浸润至黏膜下层"
B胃窦"管状腺瘤伴高级别上皮内瘤变（重度异型增生）"
C胃窦"管状腺癌，侵犯浆膜层，淋巴结见癌转移"
点击查看答案

✅ A：C16.3（胃窦恶性肿瘤）+ M8140/3（管状腺癌，恶性原发）

→ "浸润至黏膜下层"= 已突破基底膜，是恶性（/3）

✅ B：D00.2（胃原位癌）+ M8010/2（原位癌，上皮内瘤变）

→ "高级别上皮内瘤变"= 等同于原位癌，部位码在 D 章！

✅ C：C16.3（胃窦恶性肿瘤）+ M8140/3（管状腺癌，恶性原发）

→ 有淋巴结转移说明肯定是恶性，但形态学动态码仍是 /3（原发部位），不是 /6。转移信息通过 C77.- 附加编码体现

进阶题 · 综合附加编码应用 CHALLENGE

某患者从楼梯摔下，致右踝关节骨折。入院后行内固定术，术后发生切口感染，培养为金黄色葡萄球菌。出院诊断包括：

  1. 右踝关节骨折（闭合性）
  2. 术后切口感染

问题：请给出完整编码（主诊断 + 所有附加编码），并说明每个码的类型。

点击查看答案

✅ 完整编码方案：

主诊断：S82.8 踝关节骨折（闭合性）——损伤本身

附加码①：W10.- 楼梯上跌倒——外因编码（V01-Y98），说明怎么伤的

附加码②：T81.4 操作后伤口感染——手术并发症编码

附加码③：B95.6 金黄色葡萄球菌作为其他章疾病的原因——病原体附加编码

附加码④：Y83.1 外科手术作为异常反应的原因——外因编码（用于术后并发症）

💡 总结：一份病案中，4个附加码分别提供了"受伤原因""并发症性质""病原体""操作外因"四层信息。编码员必须养成"看到损伤就查外因，看到术后并发症就查 Y83/Y88，看到感染就查 B95-B98"的肌肉记忆。

✅ 八、今日重点回顾

附加编码不能做主诊断——永远是主编码的"注解"
M 编码 = 四位组织学码 + 动态码：/0良 /1未定 /2原位 /3恶原 /6转移 /9恶不肯定
肿瘤编码铁序：先形态学 → 选对肿瘤表栏 → 再查部位，不可颠倒
C/D 必须有 M 码：哪怕没做病理，也要用 M8000/3 或 M8010/3
M8140/3 ≠ 万能腺癌：子宫内膜样=M8380，乳头状=M8260，印戒=M8490
重度异型增生 = 原位癌（M8010/2），部位码用 D 章
S-T 损伤中毒码必须配 V01-Y98 外因码，不配就缺码
动态码决定 C/D 归属：C章=/3或/6，D章=/0、/1或/2
原发灶形态学永远用 /3：/6 只用于继发（转移）灶的描述
