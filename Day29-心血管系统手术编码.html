为什么重要？

心血管系统手术是中国住院病案中手术量最大、编码复杂度最高、DRG入组影响最直接的领域之一。一个PCI手术至少需要编5个码，一个瓣膜置换术可能需要3-4个码。编码缺失或错误将直接导致：

  • 入组错误：支架编码漏编36.07只编00.66 → 进入错误的ADRG
  • 权重丢失：药物洗脱支架编成非药物洗脱 → 权重差异可达30%+
  • 歧义病例（QY）：诊断与手术编码不匹配 → 无法入组

辅助手术的编码规则

体外循环（心肺转流）39.61：当心血管手术使用体外循环辅助时，必须另编码39.61。注意：

  • 冠状动脉搭桥术（CABG）几乎都使用体外循环 → 必须附加39.61
  • 单纯PCI（经皮介入）通常不使用体外循环 → 不需要39.61
  • 某些微创直视手术（MIDCAB）可能不用体外循环 → 查看手术记录确认

🫀 板块二：心脏瓣膜手术编码（35）

核心分类轴心：三层嵌套决策

心脏瓣膜手术编码的本质是一个三层嵌套决策过程，每一层都不可跨过：

1

确定手术入路（第一轴心）

闭合性（非直视）35.0 → 开放性（直视下）35.1-35.2 → 经皮介入35.96

2

确定术式类型（成形 vs 置换）

成形术（修复自身瓣膜）→ 置换术（用人工瓣膜替换）

3

定位具体瓣膜 + 假体类型（仅置换时）

主动脉瓣 / 二尖瓣 / 肺动脉瓣 / 三尖瓣 + 机械瓣 / 生物瓣

瓣膜手术编码对照表

手术入路编码说明
二尖瓣成形术 闭合性35.02闭式二尖瓣交界分离术
开放性35.12直视下二尖瓣成形术
经皮球囊35.96经皮二尖瓣球囊成形术

经导管35.05TAVI/TAVR 经导管主动脉瓣植入

✅ 正确编码思路

手术记录："全麻低温体外循环下，正中开胸，行二尖瓣机械瓣置换术"

推导：正中开胸=开放性 → 二尖瓣=瓣膜定位 → 置换=术式 → 机械瓣=假体 → 体外循环

编码：35.23（开放式二尖瓣生物瓣置换）或35.24（开放式二尖瓣机械瓣置换）+ 39.61（体外循环）

❌ 常见错误

错误一：不看手术记录直接套"二尖瓣成形术"→ 35.12，实际做了置换

错误二：将经皮球囊成形编为开放性 35.12

错误三：编了置换术却不区分机械瓣/生物瓣

错误四：忘了加编体外循环 39.61

⚠️ 关键提示：当一次手术同时涉及多个瓣膜时（如二尖瓣+主动脉瓣联合置换），各瓣膜需分别编码。例如"二尖瓣+主动脉瓣机械瓣置换"应编 35.24 + 35.22 + 39.61。

瓣膜成形术 vs 置换术：什么时候做什么？

临床上，外科医生优先尝试成形术（修复自己的瓣膜），但病变严重时必须置换。编码员不能凭"瓣膜手术"四个字就想当然编码——必须阅读手术记录中关于是否切除自体瓣膜的描述：

  • 保留自体瓣膜+缝合修复 → 成形术
  • 切除自体瓣膜+植入人工瓣膜 → 置换术
  • 球囊扩张瓣膜开口 → 经皮球囊成形术

💉 板块三：冠状动脉手术编码（36）

PCI：一个支架，五个编码

经皮冠状动脉介入治疗（PCI）是心内科最常见的操作之一。一个完整的PCI手术编码不是孤立的 36.07，而是一套"五码组合"体系：

1
支架编码（主码·必选）

查主导词"插入-支架"，根据支架是否有药物洗脱属性选择：
36.07 药物洗脱冠状动脉支架置入（DES）
36.06 非药物洗脱冠状动脉支架置入（裸金属BMS/药物涂层非抗增殖类）

2
治疗血管数量（附加码·必选）

查主导词"另编码"，统计被治疗的血管支数：
00.40 单支血管 / 00.41 两支 / 00.42 三支 / 00.43 四支及以上

3
置入支架总数量（附加码·必选）

查主导词"另编码"，统计所有支架的总个数：
00.45 1个 / 00.46 2个 / 00.47 3个 / 00.48 4个及以上
00.44 分叉血管操作（如有）

4
PTCA球囊成形（另编码·必选）

支架置入前通常先做球囊预扩张（PTCA）：
00.66 经皮冠状动脉腔内血管成形术（PTCA）

5
冠状动脉造影（附加码·必选）

主导词查"动脉造影术"，区分导管技术：
88.55 单导管技术 / 88.56 双导管技术

📋 完整实例：患者行PCI，于LAD（前降支）和LCX（回旋支）各置入1枚药物洗脱支架，共2枚，使用双导管冠状动脉造影。

完整编码：36.07 + 00.41（治疗2支血管）+ 00.46（置入2个支架）+ 00.66（PTCA）+ 88.56（双导管造影）= 共5个码

支架类型的高频陷阱：药物涂层 ≠ 药物洗脱

概念英文编码本质区别
药物洗脱支架 DES (Drug-Eluting Stent) 36.07 含抗增殖药物（雷帕霉素/紫杉醇），缓慢释放抑制内膜增生
裸金属支架 BMS (Bare Metal Stent) 36.06 纯金属，无任何药物涂层
药物涂层支架 Drug-Coated Stent 36.06 含肝素/碳涂层等抗凝/惰化材料，非抗增殖

⚠️ 关键陷阱：看到"药物涂层"就编 36.07 是高频错误！必须区分：

  • 涂层是抗增殖药物（雷帕霉素、紫杉醇等）→ 36.07
  • 涂层是肝素/碳涂层（抗凝/惰化，非抗增殖）→ 36.06

不确定时，查阅手术记录中支架的品牌型号（如进口的XIENCE、Resolute → DES=36.07；国产的BuMA等需确认说明书）。

🔬 重难点深度解析：PCI "五码"真的一个都不能少吗？

问题还原：当手术记录"信息不完整"时怎么办？

在实际编码工作中，最头疼的不是规则复杂，而是手术记录信息缺失——不写导管类型、不写支架品牌、不写是否做了PTCA预扩。但编码员不能因为"信息不全"就漏编：

场景一：支架类型不明 → 36.06 还是 36.07？

❌ 常见错误

手术记录只写"支架置入术"，不写品牌 → 编码员随意编 36.07 或 36.06

后果：如果实际是裸金属支架（BMS）却被编为 36.07（药物洗脱），DRG入组权重被抬高 → 医保飞检中的高编高套风险

✅ 正确做法

1. 查找耗材条码/收费记录确认支架品牌型号

2. 对照品牌型号确认药物洗脱属性

3. 确实无法确认时，应按保守原则编 36.06（非药物洗脱），避免高编嫌疑

4. 同时记录质控备注：待确认支架类型

场景二：PTCA预扩是否单独编码？

编码规则：支架置入术通常常规包含球囊预扩张（PTCA）这一步骤——先扩开狭窄，再放支架。ICD-9-CM-3官方编码要求另编码PTCA（00.66）。

但注意区分：

  • ✅ 常规支架置入前的球囊预扩 → 编00.66
  • ✅ 仅做球囊扩张不做支架（药物球囊DCB等） → 只编00.66，不编支架码
  • ❌ 将PTCA省略不编 → 手术信息缺失，影响DRG权重计算

场景三：治疗血管数 vs 支架数的区别

概念编码范围计数逻辑实例
治疗血管数量 00.40-00.43 统计被干预的冠脉血管支数（LAD、LCX、RCA等） LAD+LCX各治疗=2支→00.41
置入支架总数量 00.45-00.48 统计置入的支架总个数（同一血管可能有多个支架） LAD放2个+LCX放1个=共3个→00.47

💡 一句话记忆：00.4x 是"干了几个地方"，00.4x+5（即00.45-00.48）是"放了几个东西"。两者通常不等！

患者行冠状动脉支架置入术，手术记录描述："于前降支中段置入XIENCE Xpedition 3.0×28mm药物洗脱支架1枚"

点击查看答案与推理

问题A：编 36.07。

推理：XIENCE Xpedition 是雅培公司的药物洗脱支架（DES），其涂层为依维莫司（everolimus）——一种抗增殖药物，符合"药物洗脱"定义 → 编 36.07。

问题B：编 36.06。

推理：肝素涂层的作用是抗凝（防止血栓），不是抗增殖（抑制内膜增生）。ICD-9-CM-3区分支架类型的关键是"是否含抗增殖药物并控制释放"，肝素涂层不满足此条件 → 归类为非药物洗脱支架 36.06。这就是"药物涂层≠药物洗脱"的核心陷阱。

2
基础题 起搏器类型判断

三名患者分别行起搏器置入术，请判断编码：

A：III度AVB患者，经右锁骨下静脉穿刺送入临时起搏电极至右心室，用于永久起搏器置入前的过渡治疗。

B：病窦综合征患者，择期置入双腔永久起搏器（DDD模式），首次手术。

C：二尖瓣置换术中，心脏停搏后心率缓慢，置入心外膜临时起搏导线。

点击查看答案与推理

A：编 37.78（暂时性经静脉起搏器系统置入）。

推理：非手术场景下的独立紧急起搏操作 → 排除 39.64；经静脉穿刺 → 37.78。

B：编 37.83（首次置入双腔装置）。

推理：首次永久起搏+双腔（心房+心室）→ 37.83。单腔编码 37.81/37.82 仅用于仅植入一根电极导线的情况。

C：编 39.64（手术中心脏起搏器）。

推理：伴随心脏手术（瓣膜置换）中使用的临时起搏 → 这是 39.64 的典型适应场景，不能编 37.78。

3
基础题 瓣膜手术入路判断

以下三段手术记录摘要，请分别判断瓣膜手术编码：

A："全麻下，正中胸骨劈开，建立体外循环，阻断主动脉，切开右心房，经房间隔入路，直视下用成形环行二尖瓣成形术。"

B："局麻下，经皮穿刺股静脉，送球囊导管至二尖瓣口，行球囊扩张成形术。"

C："全麻下，左胸前外侧小切口（不劈胸骨），电视胸腔镜辅助下行二尖瓣成形术。"

点击查看答案与推理

A：35.12（直视下二尖瓣成形术）+ 39.61（体外循环）

推理：正中开胸+体外循环+直视下操作 → 开放性手术 → 35.1x系列。二尖瓣+成形术 → 35.12。

B：35.96（经皮瓣膜成形术——二尖瓣）

推理：经皮穿刺+球囊扩张 → 介入手术 → 不适用35.0或35.1 → 35.96。

C：35.12（直视下二尖瓣成形术）

推理：虽然是微创小切口，但依然是"直视下"进行瓣膜操作——胸腔镜提供的是视觉通道，不改变手术入路分类原则。关键是"是否直视瓣膜"，而非切口大小。

编码提示：微创胸腔镜心脏手术仍然属于"开放性"（直视下）手术范畴，编码仍为35.1x，不能编为经皮介入35.96。

4
进阶题 PCI完整编码推导

患者因"不稳定型心绞痛"行PCI手术，手术记录描述：

"经右桡动脉穿刺，采用双导管技术行冠状动脉造影。于LAD中段置入Resolute Integrity 3.5×24mm药物洗脱支架1枚，于LCX近段置入BioFreedom 3.0×18mm支架1枚，分别以球囊预扩后置入。"

请给出完整的编码组合（所有编码）。

点击查看答案与推理

完整编码（6个码）：36.07 + 00.41 + 00.46 + 00.66 + 88.56

推理过程：

  1. 支架编码：LAD用Resolute Integrity（美敦力DES，zotarolimus抗增殖）= 36.07；LCX用BioFreedom（无聚合物Biolimus涂层支架，仍属DES）= 36.07。两个都是药物洗脱 → 36.07 够用，不需要分别编36.06和36.07。
  2. 治疗血管数：LAD + LCX = 2支 → 00.41
  3. 支架总数：LAD 1枚 + LCX 1枚 = 2枚 → 00.46
  4. PTCA：明确记录"以球囊预扩" → 00.66
  5. 冠脉造影：双导管技术 → 88.56

共5个码（36.07作为主码只编一次，即使放了2个支架也是同一个36.07）。

5
进阶题 CABG + 瓣膜联合手术编码

患者行心脏多联手术，手术记录摘要：

"全麻低温体外循环下，正中胸骨劈开。取大隐静脉，行主动脉-前降支、主动脉-对角支、主动脉-钝缘支三根冠状动脉旁路移植。同时行二尖瓣机械瓣置换术。"

问题：请给出完整的编码组合。注意：是否需要另编码取大隐静脉的手术？

点击查看答案与推理

完整编码：

36.13（主动脉-冠状动脉搭桥，三根）+ 35.24（开放式二尖瓣机械瓣置换）+ 39.61（体外循环）+ 38.43（下肢静脉切除用作移植物）

推理过程：

  1. CABG：大隐静脉作为桥血管→主动脉-冠脉搭桥，搭了3根桥（前降支、对角支、钝缘支）→ 36.13
  2. 瓣膜：正中开胸+直视下+二尖瓣+机械瓣置换 → 35.24
  3. 体外循环：明确使用了体外循环 → 39.61
  4. 取大隐静脉：ICD-9-CM-3规定，当取下肢静脉用于血管移植搭桥时，需另编码血管获取术 → 38.43（下肢静脉切除）或 38.48（下肢静脉切除+置换）。考虑到该场景是取静脉用作移植物而非治疗静脉疾病，编 38.43 更准确。

共4个码。这是一个典型的"联合手术编码拼图"，每个操作都需独立体现。

6
进阶题 先心病介入+起搏器综合编码

患儿5岁，因"房间隔缺损 + 病态窦房结综合征"入院，行同期手术：

"经皮穿刺股静脉，行房间隔缺损封堵术（Amplatzer封堵器），同期经左侧锁骨下静脉穿刺置入单腔永久起搏器（VVI模式，无频率应答）。"

请给出完整编码，并说明编码决策的关键判断点。

点击查看答案与推理

完整编码：35.52（房间隔缺损伞堵修补术）+ 37.81（首次单腔装置置入，未特指节律反应）

推理过程：

  1. ASD封堵：经皮穿刺+封堵器修补房间隔缺损 → 属于闭合性心脏间隔修补术。主导词查"修补术-房间隔缺损-假体装置-闭合性"→ 35.52（房间隔缺损伞堵修补术）。不是开放性修补 35.51 或 35.61。
  2. 起搏器：永久性+首次+单腔（仅心室VVI）+无频率应答功能 → 37.81。
  3. 不需要体外循环编码：先心病介入封堵和起搏器置入均不涉及体外循环。

关键判断点：ASD封堵术的"闭合性"判断：只要不切开心脏、不在直视下操作，就属于闭合性（35.5x），不论经皮还是经胸小切口。如果手术记录描述为"直视下修补"（切开心房），则应编为开放性 35.51/35.61。
