🩸 二、糖尿病编码核心——E10-E14 完整剖析

2.1 五大分型：不是所有糖尿病都叫"糖尿病"

糖尿病的ICD-10分类轴心是病因类型而非治疗方式。很多编码员以为"打胰岛素的就是1型，吃药的就是2型"——这是完全错误的。分型依据是病因机制。

类目分型临床特征常见于
E10.- 1型糖尿病 胰岛素绝对缺乏，自身免疫破坏β细胞，C肽极低 青少年起病、酮症倾向、必须依赖胰岛素
E11.- 2型糖尿病（最高频） 胰岛素抵抗为主伴相对缺乏，或分泌缺陷为主伴抵抗 中老年起病、可服药/注射胰岛素、约90%糖尿病
E12.- 营养不良相关性糖尿病 热带地区青年发病，与蛋白质缺乏相关 罕见（中国极少使用）
E13.- 其他特指的糖尿病 继发于胰腺切除、药物、内分泌病、感染等 类固醇性糖尿病、胰腺切除后糖尿病
E14.- 未特指糖尿病 没有足够信息确定分型 残余类目，应尽量避免使用！

2.2 第四位亚目（.0-.9）：并发症决定编码精度

糖尿病编码的第四位字符是对并发症类型的区分。ICD-10规定：如果存在多个并发症，应选择最严重或最特指的那个作为第四位亚目，不能用.7（多个并发症）来回避判断。

亚目码并发症星号配对典型场景
.0昏迷（高渗性/低血糖性）—入院时意识障碍、血糖极高或极低
.1酮症酸中毒—口干、呼吸深快（Kussmaul）、血酮升高
.2†肾病N08.3*DN（糖尿病肾病）四期/五期
.3†眼病H36.0*DR（糖尿病视网膜病变）I-VI期
.4†神经病变G63.2*DPN（多发性/单发性神经病变）
.5周围循环障碍—DF（糖尿病足）、末梢血管缺血
.6其他特指并发症—糖尿病性关节病、糖尿病性皮肤病
.7多个并发症—肾病+眼病+足病同时存在时才用
.8未特指并发症—有并发症但未指明具体类型
.9无并发症—单纯血糖控制，无靶器官损害
💡 亚目选择口诀："昏迷0，酸毒1，肾病2，眼睛3，神经4，循环5，其他6，多个7"——熟记这九个数字是糖尿病编码的肌肉记忆。

2.4 "使用胰岛素"不等于"1型糖尿病"——最容易踩的坑

❌ 错误做法

患者用胰岛素 → 编码员想当然编E10（1型糖尿病）

后果：DRG入组错误，数据污染

✅ 正确做法

2型糖尿病长期使用胰岛素 → 编码 E11.- + Z92.2（长期使用胰岛素）

无论用什么治疗方式，分型看病因不看治疗

⚠️ 记忆锚点：E11类目标题本身就包含"非胰岛素依赖型糖尿病"这个别名。字面上明确告诉你：E11也可以是胰岛素依赖状态。如果现在是胰岛素治疗 ≠ 是1型糖尿病。关键是看C肽水平、自身抗体和起病年龄。

🦋 三、甲状腺疾病编码——三轴心分类法

3.1 甲状腺疾病的三个分类轴心

ICD-10对甲状腺疾病的分类有三条交叉轴线：功能状态（甲亢/甲减/正常）、形态改变（肿大/结节/弥漫）和病因（缺碘/自身免疫/药物/手术）。编码时按"先功能→再病因→最后形态"的顺序锁定。

3.2 "毒性"和"非毒性"——不是中文口语里的"有毒"

在甲状腺编码中，"非毒性"= 功能正常（甲功三项正常），"毒性"= 功能亢进。这是译名造成的常见误解。一个拳头大的甲状腺结节，只要T3/T4/TSH正常，就是"非毒性甲状腺肿"（E04.-），而不是什么"毒性"。

例1："单纯性甲状腺肿"

临床诊断：单纯性甲状腺肿
→ 查主导词"甲状腺肿" → 非毒性 → 弥漫性
→ E04.0 非毒性弥漫性甲状腺肿
→ 如果伴TSH升高 → 可能转为E03（甲减）

例2："甲状腺结节"

B超：单发结节2cm，甲功正常
→ 非毒性单结节性甲状腺肿
→ E04.101 非毒性单结节甲状腺肿（国临版）
→ 关键：甲功正常 = 非毒性

🧬 四、高脂血症与代谢疾病编码——不看化验单的话都是E78.5

4.1 高脂血症（E78）：为什么E78.5是"劣质编码"

E78.-下五个亚目的差异化来自于血脂四项的具体数值：TC（总胆固醇）、TG（甘油三酯）、HDL-C、LDL-C。如果一个编码员看到"高脂血症"四个字就直接编E78.500，等于放弃了80%的诊断信息。

亚目疾病名称判断依据占比
E78.0 纯高胆固醇血症 TC↑, TG正常；LDL-C↑为主 约35%
E78.1 纯高甘油酯血症 TG↑, TC正常 约15%
E78.2 混合型高脂血症 TC↑且TG↑ 约40%（最常见）
E78.3 高乳糜微粒血症 乳糜微粒↑, TG极度升高 罕见
E78.5 未特指的高脂血症 仅在化验单缺失时使用 应→0%
⚠️ 编码操作规范："高脂血症"入院时，首先查阅入院24h内的血脂四项报告——TC（正常 <5.2）、TG（正常 <1.7）。若患者只做了TC检验未做TG，"高脂血症"只能用E78.5——这就是质控检查会追查的问题，即实验室检查不完整。

4.2 肥胖（E66）：BMI只是起点

肥胖编码E66的四位亚目关注病因和有无低通气综合征：

  • E66.0 热量过剩性肥胖 — BMI≥28且无继发病因
  • E66.1 药物性肥胖 — 长期使用糖皮质激素、抗精神药物等
  • E66.2 极度肥胖伴肺泡低通气（Pickwickian综合征）— BMI常>40+PaCO₂升高+嗜睡
  • E66.8 其他肥胖 — 病态肥胖等特指情况
  • E66.9 未特指肥胖
💡 编码要点：单纯肥胖作为主诊断的情况很少。肥胖通常是"肥胖+高血压（I10）+糖尿病（E11）+高脂血症（E78）+睡眠呼吸暂停（G47.3）"的组合编码模式，需要仔细阅读病程记录判断哪个是住院主要治疗目的。

🔬 五、重难点深度解析：糖尿病并发症编码——"多重并发症"的致命陷阱

为什么E11.7（多个并发症）是个"高危编码"？

许多编码员看到出院诊断上有"糖尿病肾病""糖尿病视网膜病变""糖尿病足病"三个并发症，就直接编码E11.7（伴多个并发症）——认为这样"一次性覆盖了"所有问题。但这个做法存在严重隐患：

隐患一：E11.7没有星剑号配对

E11.7类目下，ICD-10没有提供对应的星号编码。这意味着如果你用E11.7来表达糖尿病肾病+眼病，你就无法添加N08.3*和H36.0*作为附加编码——这些星号是DRG并发症/合并症（CC/MCC）权重计算的重要依据。

隐患二：DRG权重损失

编码方案DRG影响权重差异
E11.2† + N08.3*（仅肾病） MCC入组（严重并发症） 基准权重 ≈ 1.0
E11.7 "多个并发症" 可能仅CC（一般并发症） 权重下降 15%-30%
E11.2† + N08.3* + E11.3† + H36.0* + E11.5（全部分开编） 多个MCC最大权重 权重最高，信息最完整

隐患三：ICD-10明确约定E11.7的使用场景极窄

第一卷关于E11.7的注释明确表述："该亚目只有在多个并发症均未达到单独作为亚目的标准，或无法确定哪一个是最主要治疗焦点时使用。"——换言之，E11.7是最后选择，不是偷懒工具。

✅ 正确编码策略：
1. 先列出所有并发症，选出与本次住院治疗目的最相关的那个作为糖尿病第四位亚目
2. 将其他并发症作为独立诊断行分别编码（不用"合并"到糖尿病主诊断的第四位里）
3. 例如：患者本次因肾透析入院 + 有DR → 用E11.2†为主体 + 另外单独加编码E11.3† H36.0*描述视网膜病变

✏️ 六、今日案例练习（共6题）

前3题为基础题，后3题为进阶题。点击答案查看完整推理过程。

1
基础题 · 糖尿病分型判断 PRACTICE

请判断以下3个糖尿病患者的正确分类类目（E10/E11/E13/E14），并说明理由：

  1. 25岁男性，C肽<0.01ng/mL，GAD抗体阳性，每日4次胰岛素注射
  2. 58岁女性，BMI 29，已口服二甲双胍5年，近半年加用长效胰岛素（来得时）
  3. 70岁男性，胰腺癌术后3个月，血糖升高需胰岛素治疗
点击查看答案

✅ 答案：

1. E10.-（1型糖尿病） — C肽极低+抗体阳性 → β细胞自身免疫性破坏 → 绝对缺乏 → 1型

2. E11.-（2型糖尿病） — 中老年起病+肥胖+前期口服药有效 → 胰岛素抵抗型。使用胰岛素不能改变分型！如有长期使用胰岛素史，加编Z92.2

3. E13.-（其他特指的糖尿病） — 胰腺切除后 → 继发性糖尿病 → 编码于E13（其他），而非E10/E11。这是最常见的继发糖尿病场景

💡 核心教训：分型看病因（血清学+病史），不看当前治疗方式。

2
基础题 · 高脂血症亚目选择 PRACTICE

某住院患者出院诊断"高脂血症"。查阅其入院24h化验报告：TC = 6.8mmol/L（↑），TG = 2.5mmol/L（↑），HDL-C = 0.9mmol/L（↓），LDL-C = 4.5mmol/L（↑）。

问题：应编E78.-的哪个亚目？为什么不能编E78.5？

点击查看答案

✅ 答案：E78.2 混合型高脂血症

推理过程：

TC = 6.8 > 5.2（↑）→ 存在高胆固醇血症
TG = 2.5 > 1.7（↑）→ 存在高甘油三酯血症
→ 两者同时升高 → E78.2混合型高脂血症

为什么不能编E78.5：化验单数据完整！E78.5是残余类目，仅当没有血脂报告时才不得已使用。有明确化验结果却编E78.5，在质控检查中属于"编码深度不足"缺陷。

💡 口诀：单TC高→E78.0 / 单TG高→E78.1 / 双双高→E78.2

基础题 · 甲状腺编码 PRACTICE

以下3个甲状腺诊断，请给出正确的ICD-10编码：

临床诊断甲功编码
Graves病（弥漫性甲亢）T3↑ T4↑ TSH↓？
甲状腺多发结节（甲功正常）T3/T4/TSH正常？
亚急性甲状腺炎（疼痛性）可正常或一过性甲亢？
点击查看答案

✅ 答案：

① 编码：E05.0 甲状腺功能亢进伴弥漫性甲状腺肿（Graves病）
主导词"甲状腺功能亢进" → 伴 → 弥漫性甲状腺肿 → E05.0。注意Graves病不是检索"Graves"而是"甲状腺功能亢进"

② 编码：E04.2 非毒性多结节性甲状腺肿
甲功正常 = "非毒性"。多结节 → E04.2。关键词"非毒性"是ICD-10译名特有概念，=功能正常

③ 编码：E06.1 亚急性甲状腺炎
主导词"甲状腺炎" → 亚急性 → E06.1。De Quervain甲状腺炎即为此编码

进阶题 · 糖尿病星剑号完整推导 CHALLENGE

患者，女，62岁。"2型糖尿病"病史12年。本次因双下肢麻木、视力模糊入院。入院检查：空腹血糖16.2、眼底检查：双眼糖尿病性视网膜病变III期、肌电图：糖尿病性多发性周围神经病变。出院诊断："2型糖尿病伴视网膜病变""2型糖尿病伴周围神经病变"。

问题：

  1. 应该编一个糖尿病码还是两个？
  2. 星剑号（†/*）怎么配？
  3. 主诊断应该选哪个？
点击查看答案

✅ 答案：

1. 应编两个糖尿病码：一个用于眼病，一个用于神经病变。不能用E11.7（多个并发症）替代——会丢失星号编码无法反映DRG权重。
但本次住院需要判断主要治疗的焦点：视力模糊和下肢麻木都是主诉。如果眼底激光治疗是主要操作 → 以眼病为主诊。

2. 编码组合：
① E11.300† 2型糖尿病性视网膜病变（剑号，主诊断）
   H36.000* 糖尿病性视网膜病变（星号，附加）
② E11.400† 2型糖尿病性周围神经病变（剑号，附加诊断）
   G63.200* 糖尿病性多神经病变（星号，附加）

3. 主诊断：E11.300†（2型糖尿病性视网膜病变）
依据：DRG入组规则下，选择对本次医疗资源消耗影响最大的诊断。视网膜病变通常匹配的手术操作（激光等）会驱动入组变化。

💡 进阶要点：两个星号（H36.0*和G63.2*）都要填在"其他诊断"中。虽然主诊断只有一个，但附加编码能完整描述患者的疾病负荷（complication burden），直接影响DRG并发症等级的判定。

进阶题 · E章 vs M章交叉判断 CHALLENGE

患者，男，55岁。因"右足第一跖趾关节红、肿、热、痛3天"入院。实验室检查：血尿酸 580μmol/L（↑）。关节液检查见尿酸盐结晶。既往有2型糖尿病史。出院诊断："痛风性关节炎（急性发作）""2型糖尿病""高尿酸血症"。

问题：

  1. 哪些诊断应该合并编码？哪些应该分开？
  2. 主诊断应该是什么？
  3. 给出完整的诊断编码列表。
点击查看答案

✅ 答案：

1. 编码逻辑：
• 痛风性关节炎+高尿酸血症 → 痛风性关节炎已包含高尿酸血症的病因学关联，高尿酸血症不应再单独编码。（注意：只有单纯高尿酸血症、没有关节炎时才用E79.0）
• 痛风性关节炎 → M章（骨骼肌系统）→ M10.-
• 这是E章和M章的典型交叉：虽然代谢病在E章，但已出现器官并发症的疾病优先归入相应系统章。

2. 主诊断：M10.002 原发性痛风（第一跖趾关节）
本次住院的主因是痛风急性发作，代谢病（糖尿病）是伴随诊断。

3. 完整编码列表：
主诊断：M10.002 原发性痛风性关节炎（第一跖趾关节）
其他诊断：E11.900 2型糖尿病（无并发症）
不编：E79.0（高尿酸血症）——已合并入M10

进阶题 · "不包括"规则辨识 CHALLENGE

以下诊断各给出一个编码建议，但其中有3个错误。请找出错误的，说明原因并给出正确编码。

#临床诊断建议编码对/错？
A甲状腺术后甲减E03.8？
B2型糖尿病，长期用胰岛素E10.9？
C糖尿病性白内障E11.3†+H28.0*？
D新生儿低血糖E16.2？
E类固醇性糖尿病E11.9？
点击查看答案

✅ 错误的有A、B、D、E（其实是4个错误！）：

A ×：甲状腺术后甲减 → 应编 E89.0（操作后甲状腺功能减退症）。
E03是"原发性"甲减。手术导致的属于"操作后内分泌和代谢紊乱"（E89），不在E00-E07中！第一卷E89明确包含了"操作后甲状腺功能减退"。

B ×：2型糖尿病长期用胰岛素 → 应编 E11.9 + Z92.2。
用胰岛素≠1型糖尿病！分型不变，加Z92.2说明用药史。

C ✓：糖尿病性白内障编码 E11.3†+H28.0* 正确。注意糖尿病性白内障和糖尿病性视网膜病变的星号不同：白内障是H28.0*，视网膜病变是H36.0*。

D ×：新生儿低血糖 → 应编 P70.4（新生儿其他低血糖症）。
属于第十六章"起源于围生期的某些情况"，而不是E章的"葡萄糖调节异常"。

E ×：类固醇性糖尿病 → 应编 E13.-（其他特指的糖尿病）。
E13是"药物/化学物诱发糖尿病"的正确归属。E11是原发性2型，不能用于继发。

✅ 七、今日重点回顾

糖尿病分型看病因：自身免疫→E10, 抵抗型→E11, 继发→E13，不能以治疗方式反推
第四位亚目代表当前住院最需要治疗的并发症，不是"什么都不挑"的.7
星剑号不嫌多：每个有星号的并发症都应单独编码，E11.7=放弃权重
高脂血症E78：必须有血脂化验 → TC和TG的具体值决定.0/.1/.2，不能默认.5
"非毒性"=功能正常：这是ICD-10专有译法，不是普通意义上的"无毒"
术后甲减≠E03：属操作后内分泌紊乱E89.0，不在甲状腺本身编码段
痛风关节炎→M10，高尿酸血症(无关节炎)→E79.0。这是E-M交叉的典型
新生儿低血糖→P70；围生期章优先，不能用E16的成人编码
