📖 一、I 章循环系统疾病全景概览

1.1 I00-I99 章节结构总览

循环系统疾病（Diseases of the circulatory system）是 ICD-10 第九章，码位范围 I00-I99。这一章涵盖了心脏、血管疾病的几乎所有类型，是病案编码中出现频率最高的章节之一。理解本章的分类结构，是编码员的基本功。

码位范围板块名称核心疾病编码频率
I00-I09风湿性热和风湿性心脏病风湿热、风湿性瓣膜病中
I10-I15高血压性疾病 ★高频原发性高血压、高血压性心脏病/肾病极高
I20-I25缺血性心脏病 ★高频心绞痛、急性心梗、冠心病极高
I26-I28肺循环疾病肺栓塞、肺动脉高压中
I30-I52其他心脏疾病心包炎、心肌炎、心内膜炎、心力衰竭高
I60-I69脑血管疾病 ★高频脑出血、脑梗死、脑血管病后遗症极高
I70-I79动脉、小动脉和毛细血管疾病动脉粥样硬化、主动脉瘤中
I80-I89静脉、淋巴结疾病静脉血栓、静脉曲张中
I95-I99循环系统其他疾患低血压、直立性低血压低
今日聚焦三大板块：
① I10-I15 高血压性疾病——"合并编码"的典型代表
② I20-I25 缺血性心脏病——心绞痛、心梗、冠心的编码核心
③ 心梗时间轴——I21→I22→I25.2 的动态转换
脑血管疾病（I60-I69）留到 Day 19 专题讲解。

1.2 I 章编码的通用规则

I 章有几条贯穿始终的编码规则，必须牢记：

1

高血压合并编码的"推定关系"

当高血压与心脏病/肾病同时存在时，ICD-10 推定存在因果关系，应使用合并编码 I11/I12/I13，而非分别编码。但继发性高血压 I15 例外。

2

缺血性心脏病的"时间轴"分类

心梗以 4 周（28 天）为界：≤28 天 = 急性（I21）；出院后 28 天内再发 = 随后性（I22）；>28 天 = 慢性（I25.8）；已愈合无症状 = 陈旧性（I25.2）。

3

同时提及高血压的附加编码

I20-I25 类目下注明"包括同时提及的高血压（I10-I15）"，需要时可使用附加编码标明存在的高血压。但若使用了 I11-I13 合并编码，则不再另编 I10。

🩸 二、高血压编码规则（I10-I15）——合并编码的核心

2.1 高血压分类总览

高血压（Hypertension）是心血管疾病中最常见的慢性病，几乎所有中老年患者病案中都会出现。ICD-10 将高血压分为原发性和继发性两大类，并对高血压导致的心脏/肾脏损害提供了合并编码。

编码名称含义使用条件
I10原发性高血压无明确病因的高血压未提及心/肾损害
I11.0高血压性心脏病伴心力衰竭高血压导致心脏病 + 心衰因果关系明确
I11.9高血压性心脏病不伴心力衰竭高血压导致心脏病，无心衰因果关系明确
I12.0高血压性肾病伴肾衰竭高血压导致肾病 + 肾衰因果关系明确
I12.9高血压性肾病不伴肾衰竭高血压导致肾病，无肾衰因果关系明确
I13.x高血压性心脏和肾病同时累及心脏和肾脏因果关系明确
I15.x继发性高血压由其他疾病引起的高血压有明确原发病因

2.2 合并编码的判断流程

高血压合并编码是考试必考的知识点，核心在于判断高血压与靶器官损害之间是否存在因果关系。

高血压 + 心脏病
→
因果关系明确？
是 → I11.x 合并编码
否 → I10 + 心脏病分别编码
高血压 + 肾病
→
因果关系明确？
是 → I12.x 合并编码
否 → I10 + 肾病分别编码
⚠️ 关键区分：I11-I13 是原发性高血压导致心/肾损害的合并编码。如果是继发性高血压（如肾血管性高血压 I15.0），则不使用 I11-I13，而应编 I15.x 并附加靶器官损害编码。

2.3 继发性高血压 I15 的病因追查

继发性高血压意味着高血压是其他疾病的症状之一，编码时必须追查原发病因。原发病因编在主诊断，I15 作为附加编码。

编码病因编码路径
I15.0肾血管性（肾动脉狭窄）主导词：高血压 → 继发性 → 肾血管性 → I15.0
I15.1继发于其他肾疾患主导词：高血压 → 继发性 → 由于 → 肾脏疾患 NEC → I15.1
I15.2继发于内分泌疾病主导词：高血压 → 继发性 → 由于 → 内分泌疾患 → I15.2
I15.8其他继发性药物引起、主动脉缩窄等
I15.9未特指的继发性应尽量避免使用

例：肾血管性高血压

临床诊断：肾动脉狭窄伴高血压

主诊断：I70.1 肾动脉粥样硬化

附加编码：I15.0 肾血管性高血压

注意：原发病因为主诊断

例：原发性高血压+心衰

临床诊断：高血压性心脏病伴心力衰竭

主诊断：I11.0

附加编码：心衰类型码（如 I50.9）

注意：不再另编 I10

❤️ 三、冠心病与心绞痛编码（I20-I25）

3.1 缺血性心脏病 I20-I25 结构

缺血性心脏病（Ischemic Heart Disease, IHD）是冠状动脉狭窄或闭塞导致心肌缺血的一组疾病，临床统称"冠心病"。ICD-10 将其分类于 I20-I25，按临床表现形式和时间进行分类。

编码名称分类轴心临床对应
I20心绞痛临床分型稳定型/不稳定型/变异型
I21急性心肌梗死部位 + 透壁性STEMI / NSTEMI
I22随后性心肌梗死部位 + 透壁性28 天内再发心梗
I23急性心梗后近期并发症并发症类型心包积血、心脏破裂等
I24其他急性缺血性心脏病—急性冠脉综合征（未分型）
I25慢性缺血性心脏病具体类型冠心病、陈旧性心梗、缺血性心肌病

3.2 心绞痛 I20 的分型编码

心绞痛（Angina Pectoris）是冠脉供血不足引起的发作性胸痛。ICD-10 按临床分型进行编码，住院患者必须找到准确分型，不能笼统编 I20.9。

编码名称临床特征编码路径
I20.0不稳定型心绞痛新发/加重/休息时发作，梗死前状态心绞痛 → 不稳定 → I20.0
I20.1伴有确证痉挛的心绞痛变异型/Prinzmetal心绞痛，冠脉痉挛心绞痛 → 伴有确证的痉挛 → I20.1
I20.8其他形式的心绞痛稳定型/劳力型心绞痛心绞痛 → 其他 → I20.8
I20.9心绞痛，未特指类型不明时才使用心绞痛 → I20.9
⚠️ 常见错误：临床经常只写"心绞痛"而不注明分型。编码员必须查阅病程记录、心电图结果，尽量找到准确分型。变异型心绞痛（I20.1）有ST段一过性抬高，不稳定型心绞痛（I20.0）有梗死前状态描述——这些临床线索是编码的依据。

3.3 慢性缺血性心脏病 I25

I25 是慢性缺血性心脏病的总称，包含冠心病、陈旧性心梗、缺血性心肌病等多个亚型，是编码工作中最容易混淆的区域之一。

编码名称使用场景能否做主诊断
I25.1冠状动脉粥样硬化性心脏病冠脉粥样硬化确诊，无急性发作可以（但应优先用更明确分型）
I25.2陈旧性心肌梗死心梗已愈合（>28天），无症状不能做主诊断（仅做附加）
I25.5缺血性心肌病冠心病导致心脏扩大、心衰可以（心衰作为附加编码）
I25.6隐匿性冠心病无症状性心肌缺血可以
I25.8其他慢性缺血性心脏病慢性心梗（>28天仍有症状）可以

❌ 常见错误

患者急性心梗出院后 3 个月因"胸闷"入院，编码员直接编 I25.1 冠心病——忽略了急性心梗病史，应为 I25.2 陈旧性心梗

✅ 正确做法

查阅既往病史：心梗 >28 天且无症状 → I25.2；心梗 >28 天仍有缺血症状 → I25.8；仅有冠脉硬化无心梗史 → I25.1

🔥 四、急性心肌梗死编码（I21-I22）——双轴心分类

4.1 急性心梗 I21 的双轴心分类

急性心肌梗死（Acute Myocardial Infarction, AMI）是 I 章编码的重中之重。ICD-10 对 I21 采用双轴心分类：第一轴心是发病部位（前壁/下壁/其他部位），第二轴心是透壁性（STEMI 透壁 vs NSTEMI 非透壁）。

编码名称对应临床心电图特征
I21.0急性前壁透壁性心梗前壁 STEMIV1-V4 ST段抬高
I21.1急性下壁透壁性心梗下壁 STEMIII/III/aVF ST段抬高
I21.2其他部位的透壁性心梗侧壁/后壁等 STEMI对应导联ST抬高
I21.3未特指部位的透壁性心梗STEMI 部位不明ST抬高但未记录部位
I21.4急性非透壁性心梗NSTEMI / 心内膜下心梗ST-T动态演变，无ST持续抬高
I21.9急性心肌梗死，未特指类型不明时使用—
国临版扩展：国临版在 I21 基础上进一步扩展，如 I21.001（急性前壁STEMI）、I21.101（急性下壁STEMI）、I21.401（急性NSTEMI）等。填报时应编到最细扩展码。

4.2 心梗时间轴：I21 → I22 → I25.2

心肌梗死的编码随时间推移而变化，这是 I 章编码的核心逻辑之一。28 天（4 周）是关键时间节点。

发病 ≤28天
→
I21 急性心梗
28天内再发新心梗
→
I22 随后性心梗（+I21同时使用）
>28天仍有症状
→
I25.8 慢性心梗
>28天已愈合无症状
→
I25.2 陈旧性心梗

I22 随后性心肌梗死的使用要点：

  • 条件：首次急性心梗后 28 天内再发新的急性心梗
  • 必须同时使用 I21：I22 与 I21 配合使用，I22 表示"本次再发"，I21 表示"首次发作"
  • 排序规则：如果本次入院主要治疗再发心梗，则 I22 为主诊断，I21 为附加诊断
  • 限制：I22 仅适用于 Type 1 心梗的再发；Type 2 心梗再发只用 I21.A1，不用 I22

4.3 心梗并发症 I23-I25 的时间界限

心梗并发症的编码与发生时间密切相关，近期并发症和远期并发症归入不同类目。

编码名称时间界限典型并发症
I23.x急性心梗后近期并发症心梗后 28 天内心包积血、心脏破裂、室间隔穿孔、乳头肌断裂
I25.x心梗后远期并发症心梗后 >28 天心室壁瘤（I25.3）、冠状动脉瘤（I25.4）
I31.x心梗后晚期心包疾患心梗后 >28 天心包积血（慢性）
I51.x心梗后其他晚期并发症心梗后 >28 天血栓形成、乳头肌断裂（慢性）
⚠️ 编码顺序：当心梗（I21/I22）和并发症（I23）同时存在时，I21/I22 为主诊断，I23 为附加诊断。但如果入院目的就是治疗并发症，则可能以并发症为主诊断（需根据具体情况判断）。

💡 五、重难点深度解析：高血压合并编码的因果判断

高血压合并编码是 I 章编码中最容易出错的区域。核心难点在于：什么时候该用合并编码（I11/I12/I13），什么时候该分别编码（I10 + 心脏病/肾病）？

5.1 合并编码的三个判断条件

条件说明示例
① 存在原发性高血压 必须是原发性（I10），继发性高血压不用合并编码 I10 原发性高血压 + I50 心衰 → 可能需要 I11.0
② 存在靶器官损害 心脏损害（心衰、心室肥厚等）或肾脏损害（肾衰、蛋白尿等） 心力衰竭、慢性肾脏病
③ 因果关系明确 病历中明确描述"高血压性心脏病"或"高血压导致肾损害" 临床诊断写"高血压性心脏病"

5.2 国临版的"推定关系"规则

在国临版编码规则中，当高血压与心脏病/肾病同时存在时，推定存在因果关系，默认使用合并编码。但有一个重要例外：

❌ 错误理解

只要患者同时有高血压和心脏病，就用 I11.x → 忽略了继发性高血压的例外

例：肾血管性高血压 + 心衰 → 错误地编为 I11.0

✅ 正确规则

推定关系仅适用于原发性高血压。继发性高血压（I15）不使用合并编码

例：肾血管性高血压 + 心衰 → 编 I15.0 + I50.x（分别编码）

5.3 高血压合并编码决策树

高血压 + 心脏病/肾病
是继发性高血压？
是 → I15.x + 靶器官码
↓ 否
原发性高血压
累及心脏？→ I11.x
累及肾脏？→ I12.x
同时累及？→ I13.x
仅高血压无损害？→ I10
⚠️ 特别注意：使用了 I11/I12/I13 合并编码后，不再另编 I10。合并编码已经包含了高血压成分。但如果有心衰，需要附加心衰类型码（I50.x）以标明心衰的具体类型。

5.4 高血压与脑血管病/冠心病的关系

高血压合并编码 I11-I13 仅涵盖心脏和肾脏两个靶器官。高血压导致的脑血管病（如高血压性脑出血）和冠心病（如急性心梗）没有合并编码，必须分别编码。

场景错误做法正确做法
高血压 + 脑出血 编 I11.x（错误：脑不是I11的靶器官） I61.x（脑出血）+ I10（高血压，附加编码）
高血压 + 急性心梗 编 I11.x（错误：心梗不属于I11范畴） I21.x（急性心梗）+ I10（高血压，附加编码）
高血压 + 心脏扩大+心衰 I10 + I50.x（分别编码） I11.0（合并编码）+ I50.x（心衰类型）

1
基础题 · 急性心梗编码 PRACTICE

患者男性，58岁，因"突发胸痛3小时"入院。心电图示 V1-V4 导联 ST 段弓背向上抬高，肌钙蛋白升高。临床诊断：急性前壁ST段抬高型心肌梗死。

问题：请给出最终编码及完整索引路径。

点击查看答案

✅ 编码：I21.0（急性前壁透壁性心肌梗死），国临版扩展码 I21.001

✅ 索引路径：梗死 → 心肌 → 急性 → 透壁 → 前壁 → I21.0

✅ 解析：STEMI = 透壁性心梗，V1-V4对应前壁，因此编 I21.0。双轴心分类：第一轴心=部位（前壁），第二轴心=透壁性（STEMI）。

2
基础题 · 高血压合并编码 PRACTICE

患者女性，72岁，有高血压病史10年。本次因"活动后气促1月"入院，超声心动图示左心室扩大、EF 38%，NT-proBNP 显著升高。临床诊断：高血压性心脏病伴心力衰竭。

问题：主诊断和附加诊断分别编什么？是否需要另编 I10？

点击查看答案

✅ 主诊断：I11.0（高血压性心脏病伴心力衰竭）

✅ 附加编码：I50.9（心力衰竭，未特指）——标明心衰类型

✅ 不再另编 I10：合并编码 I11.0 已包含高血压成分，重复编码 I10 是错误。

💡 索引路径：高血压 → 心脏病 → 伴有心力衰竭 → I11.0

3
基础题 · 心绞痛分型编码 PRACTICE

患者男性，65岁，反复夜间睡眠中发作胸痛，伴ST段一过性抬高。冠脉造影示右冠状动脉痉挛。临床诊断：变异型心绞痛。

问题：应编什么编码？为什么不能编 I20.0（不稳定型心绞痛）？

点击查看答案

✅ 编码：I20.1（心绞痛伴有确证的痉挛）

✅ 索引路径：心绞痛 → 伴有确证的痉挛 → I20.1

✅ 不能编 I20.0 的原因：变异型心绞痛（Prinzmetal）的发病机制是冠状动脉痉挛，而非冠脉固定狭窄导致的不稳定斑块。虽然变异型心绞痛在临床上属于不稳定型心绞痛的一种特殊类型，但 ICD-10 为其单独设立了 I20.1 编码，应优先使用更特异的编码。

4
进阶题 · 心梗时间轴编码 CHALLENGE

患者男性，55岁。3周前因"急性前壁心梗"在某院行PCI支架植入术。本次因"再发胸痛2小时"入院，心电图示下壁导联ST段抬高，肌钙蛋白升高。诊断：急性下壁心肌梗死（再发）。

问题：本次住院的主诊断和附加诊断分别是什么？说明理由。

点击查看答案

✅ 主诊断：I22.1（随后性下壁透壁性心肌梗死）

✅ 附加诊断：I21.0（急性前壁透壁性心肌梗死——首次发作）

✅ 理由：首次心梗后 28 天内（3周 = 21天）再发新的急性心梗，使用 I22。I22 必须与 I21 配合使用——I22 表示本次再发心梗，I21 表示首次心梗。本次入院主要治疗再发心梗，故 I22 为主诊断。

💡 索引路径：梗死 → 心肌 → 随后性 → 透壁 → 下壁 → I22.1

5
进阶题 · 继发性高血压编码 CHALLENGE

患者男性，11岁。因"血压升高伴蛋白尿6月"入院。既往肾病综合征病史2年。肾穿刺活检示 IgA 肾病。血压 160/100mmHg，口服降压药效果不佳。临床诊断：难治性高血压、IgA 肾病。

问题：高血压应编 I10 还是 I15？主诊断和附加编码如何排列？

点击查看答案

✅ 高血压编码：I15.1（继发于其他肾疾患的高血压），不是 I10

✅ 主诊断：I15.1（继发性高血压）——或以 IgA 肾病 N02.8 为主诊断，视入院目的而定

✅ 附加编码：IgA 肾病编码

✅ 理由：患者有明确的肾脏疾病基础（肾病综合征/IgA肾病），高血压是由肾脏疾病导致的——属于继发性高血压。虽然临床写"难治性高血压"，但编码员必须根据病因判断：有肾脏疾患基础 → 继发性 → I15.1。

💡 索引路径：高血压 → 继发性 → 由于 → 肾脏疾患 NEC → I15.1

进阶题 · 心梗并发症编码 CHALLENGE

患者男性，68岁。急性前壁心梗后第5天，突发胸痛加重，超声心动图示心包积液（中等量），考虑心梗后心包炎。同期还发现室间隔穿孔。

问题：请列出所有应编的疾病编码及排列顺序。

点击查看答案

✅ 编码列表：

① I21.0（急性前壁透壁性心肌梗死）——主诊断

② I23.0（心包积血作为急性心梗后近期并发症）——附加诊断

③ I23.1（室间隔穿孔作为急性心梗后近期并发症）——附加诊断

✅ 理由：心梗后第5天（28天内）出现的并发症属于近期并发症，分类于 I23。心包炎/心包积液和室间隔穿孔都是 I23 的亚目。I21 为主诊断，I23 为附加诊断。

💡 关键点：如果这些并发症在心梗 28 天后才发现，则不编 I23，而编 I31（心包疾患）或 I51（其他晚期并发症）。

✅ 七、今日重点回顾

I章结构：I10-I15高血压、I20-I25缺血性心脏病、I60-I69脑血管病为三大高频板块
高血压合并编码：I11（心）、I12（肾）、I13（心+肾），使用后不再另编I10
继发性高血压I15不使用合并编码，需追查原发病因
急性心梗I21双轴心：部位（前壁/下壁/其他）× 透壁性（STEMI/NSTEMI）
心梗时间轴：≤28天I21 → 28天内再发I22 → >28天I25.8 → 愈合I25.2
I22必须配合I21使用：排序视本次入院目的决定
心绞痛I20分型：不稳定I20.0、痉挛I20.1、稳定I20.8，尽量避免I20.9
高血压合并编码仅涵盖心脏和肾脏，脑出血/心梗需分别编码
