🗺️ 一、G章十一大板块全景速览

G章（神经系统疾病）涵盖从中枢神经系统到周围神经、从变性病到发作性疾病的全部谱系。与临床教材的"神经系统疾病"概念不同，ICD-10将脑血管疾病归入I章（I60-I69），将精神行为障碍中器质性部分归入F章（F00-F09）。因此编码员必须时刻注意"不包括"注释。

板块码位代表性疾病编码特征
① 炎症性疾病G00-G09 细菌性脑膜炎 G00、病毒性脑膜炎 G02.0*、脑炎 G04、颅内脓肿 G06 感染性脑膜炎大部分为星号码（病原体编码为剑号，如A39.0†+G01*）
② 系统性萎缩G10-G14 亨廷顿病 G10、遗传性共济失调 G11、脊髓性肌萎缩 G12、ALS G12.2 全部为原发性神经退行性疾病，无星号编码
③ 锥体外系和运动障碍G20-G26 帕金森病 G20、继发性帕金森 G21、肌张力障碍 G24 G20 vs G21的区分是核心考点
④ 其他变性病G30-G32 阿尔茨海默病 G30†、额颞叶痴呆 G31.0 星剑号系统核心阵地：G30† + F00*-F03*
⑤ 脱髓鞘疾病G35-G37 多发性硬化 G35、视神经脊髓炎 G36.0、急性播散性脑脊髓炎 G04.0 G37亚目区分不同脱髓鞘类型
⑥ 发作性和阵发性疾病 ★G40-G47 癫痫 G40、偏头痛 G43、TIA G45、睡眠障碍 G47 高频考点区，癫痫分型 + TIA鉴别最重要
⑦ 神经/神经根/神经丛G50-G59 三叉神经痛 G50.0、面神经炎 G51.0、腕管综合征 G56.0 以解剖定位为核心轴心
⑧ 多神经病和周围神经G60-G64 遗传性神经病 G60、吉兰-巴雷综合征 G61.0、糖尿病性神经病变 G63.2* G63全部为星号，表示分类于他处的疾病引起的
⑨ 肌神经接头和肌肉G70-G73 重症肌无力 G70.0、肌营养不良 G71.0 G73* 为分类于他处的肌病
⑩ 脑瘫和瘫痪综合征G80-G83 脑性瘫痪 G80、偏瘫 G81、截瘫 G82 不包括中风后遗症（I69）
⑪ 其他神经系统疾病G90-G99 自主神经病变 G90、脑积水 G91、操作后并发症 G97 混合收纳各种未归类的神经系统问题
💡 记忆口诀："炎萎锥变脱，阵发神多肌，瘫痪其他收" —— 11个字对应11个板块：炎(G00)→萎(G10)→锥(G20)→变(G30)→脱(G35)→阵(G40)→神(G50)→多(G60)→肌(G70)→瘫痪(G80)→其他(G90)。

⚡ 二、G章 vs I章生死线：TIA vs 脑梗死 —— 一字之差，天壤之别

这是G章编码中出错率最高的分水岭。TIA（短暂性脑缺血发作）编码在G45，脑梗死编码在I63。但临床上医生经常混写"TIA"和"腔隙性脑梗死"，编码员若不核对影像报告，就会直接入错组。

2.1 TIA和脑梗死的编码对照

临床诊断影像学依据症状持续时间编码所属章
TIA（短暂性脑缺血发作） MRI-DWI 无急性梗死灶 通常 < 24小时，且不留后遗症 G45.9 G章
椎基底动脉TIA 同上 同上 G45.0 G章
颈内动脉TIA 同上 同上 G45.1 G章
腔隙性脑梗死 MRI-DWI 有急性梗死灶 可能持续 > 24小时或有后遗症 I63.801 I章
脑梗死（未特指） CT/MRI证实梗死灶 持续存在 I63.9 I章

2.2 关键鉴别流程

1

看影像报告——DWI是金标准

MRI-DWI高信号 = 急性梗死灶 → I63；DWI阴性 = 无梗死 → G45

CT在超早期（<6小时）可能无法显示梗死灶，此时不能仅凭CT阴性就编G45

2

问症状持续时间

症状在24小时内完全消失且无影像学梗死证据 → G45

症状持续 > 24小时或有后遗症 → 几乎不是TIA，考虑I63

3

核对颅内血管筛查结果

MRA/CTA提示大动脉狭窄（颈内动脉/大脑中动脉≥50%）→ 即使DWI阴性也高度关注

心源性栓塞证据（房颤+突发神经症状）→ 查超声心动，可能影响TIA的亚分类

2.3 四个最高频误编场景

❌ 错误：医生写"TIA"，编码员直接编G45

真相：影像报告显示DWI高信号（急性腔隙性梗死）→ 应编I63.801，而非G45

✅ 正确：必须核影像

无论诊断怎么写，DWI有高信号就是脑梗死。编码以客观检查结果为准，不以临床印象为准

❌ 错误：医生写"脑梗死"，编码员直接编I63

真相：医生有时会把症状持续<1小时的TIA也写成"脑梗死"，但DWI阴性 → 应编G45

✅ 正确：必须看症状时长和DWI

症状已完全恢复 + DWI阴性 → 不论医生写什么，编码都应该是G45

⚠️ 实战铁律：TIA和脑梗死的编码区分，不能单凭临床诊断文字决定。必须同时核对三样东西——①症状持续时间、②MRI-DWI是否阳性、③是否有神经功能缺损后遗症。三缺一，编码就有风险。

⚡ 三、癫痫编码（G40-G41）—— 分型不清就是隐性错误

3.1 为什么分型如此重要？

癫痫是G章中最复杂的亚目分支之一。ICD-10将癫痫按照发作类型（全身性/局灶性/未分类）和是否伴有癫痫持续状态两条轴心进行分类。编码员如果只关注"癫痫"这个笼统诊断而忽略亚型，不仅直接影响DRG入组精度，还会导致临床统计失真。

3.2 G40癫痫分类全表

亚目名称核心特征典型场景
G40.0 局灶性（部分性）特发性癫痫 起源于一侧大脑半球，无意识丧失 儿童良性中央-颞区棘波癫痫（BECT）
G40.1 局灶性症状性癫痫 由已知脑部病变引起的部分性发作 脑卒中后癫痫、脑外伤后癫痫
G40.2 局灶性症状性癫痫 + 复杂部分性发作 部分性发作伴意识障碍（常来自颞叶） 最高频编码：成人癫痫大部分为此型
G40.3 全身性特发性癫痫 ★ 双侧大脑同时受累，意识丧失 失神发作、青少年肌阵挛癫痫
G40.4 其他全身性癫痫 婴儿痉挛、Lennox-Gastaut等特殊综合征 难治性儿童癫痫
G40.5 特殊癫痫综合征 反射性癫痫、阅读性癫痫等 低频特殊类型
G40.6 癫痫大发作（未特指） 传统用语，对应全身强直-阵挛发作 不建议单独使用——尽量细分子G40.3
G40.7 癫痫小发作（未特指） 传统用语，对应失神发作 不建议单独使用
G40.8 其他癫痫 未归类于以上亚目的癫痫 混合型癫痫
G40.9 未特指的癫痫 无具体分型信息 仅作为最后选择

3.3 G41 癫痫持续状态——独立编码

癫痫持续状态（Status Epilepticus）在ICD-10中被单独列为G41类目，不能混在G40中。这是一个关键区别：

亚目名称判定标准
G41.0癫痫大发作持续状态强直-阵挛发作 > 5分钟，或连续发作间期意识未恢复
G41.1癫痫小发作持续状态失神状态持续 > 15分钟
G41.2复杂部分性癫痫持续状态局灶性发作伴意识障碍持续状态
G41.8其他癫痫持续状态肌阵挛持续状态等
G41.9未特指的癫痫持续状态无进一步信息
⚠️ 关键区别：
  • 患者因一次癫痫发作入院，非持续状态 → 编G40.x
  • 患者因癫痫持续状态入院（发作 > 5分钟或连续发作间不清醒）→ 编G41.x
  • 不能同时编G40和G41——持续状态本身就意味着这是特殊类型的发作，优先使用G41

3.4 常见错误：G40.3 vs G40.2 混淆

❌ 看到"癫痫"就编G40.9

这是最懒惰的编码方式。即使医生未写明分型、病历中未记载脑电图结果，也应向临床医生追问发作类型

✅ 四步判别法

①看脑电图：双侧同步棘慢波 → 全身性(G40.3)
②看意识：发作时意识清楚 → 局灶性(G40.0/1/2)
③看影像：有无脑器质病灶 → 症状性(G40.1/2)
④看时长：>5分钟连续发作 → G41

🔄 四、帕金森的编码迷宫：G20 vs G21 vs G31.8†

帕金森综合征是编码考试的高频陷阱——同一组症状，因病因不同而编码完全不同。

4.1 三个帕金森编码的鉴别表

编码名称病因典型场景L-DOPA反应
G20 帕金森病（原发性） 黑质多巴胺能神经元不明原因退行性变 60岁以上起病，不对称起病，静止性震颤 ✅ 反应良好
G21.0 恶性综合征（抗精神病药致） 抗精神病药物阻断多巴胺受体 使用氟哌啶醇/氯丙嗪后出现的锥体外系症状 ❌ 反应差
G21.1 其他药物性帕金森 利血平、胃复安（甲氧氯普胺）、钙通道阻滞剂等 有明确药物使用史，停药后可逆 ❌ 反应差
G21.2 继发性帕金森（创伤性） 头部外伤后 拳击手脑病（拳击手帕金森） ❌ 可变的
G21.3 脑炎后帕金森 病毒性脑炎后（如昏睡性脑炎） 1917-1928年大流行的"昏睡病"后遗症 ❌ 可变的
G21.4 血管性帕金森综合征 多发腔隙性脑梗死累及基底节 下肢受累为主（"下半身帕金森"），常伴锥体束征 ❌ 反应差
G31.8† + F02.3* ★ 路易体痴呆（DLB）★ α-突触核蛋白沉积 帕金森症状+ 波动性认知障碍 + 视幻觉——三核心症状 反应中等

4.2 编码决策逻辑

患者有帕金森症状（震颤+强直+运动迟缓） │ ├─ 是否使用过抗精神病药？ │ YES → G21.0（恶性综合征）或 G21.1（其他药物性） │ ├─ 是否有明确的血管病因（多发性脑梗死）？ │ YES → G21.4（血管性帕金森综合征） │ ├─ 是否伴有波动性认知障碍 + 视幻觉？ │ YES → G31.8† + F02.3*（路易体痴呆） │ ├─ 是否不对称起病 + L-DOPA反应良好 + 无已知继发病因？ │ YES → G20（原发性帕金森病） │ └─ 其他明确原因（创伤、脑炎后、中毒等） → G21.2/G21.3/G21.8/G21.9
⚠️ 关键认知：G21.4（血管性帕金森综合征）在临床中远比想象中常见——很多脑小血管病患者同时有步态障碍和锥体外系症状。如果编码员不仔细看病历中的影像报告（有无多发腔梗），就很容易误编成G20。而G20与G21.4的DRG入组路径完全不同。

🔗 五、阿尔茨海默病编码：G30† 主体 + F00*~F03* 表现——星剑号联动

5.1 为什么AD需要两个码？

阿尔茨海默病不仅是G章的核心疾病，更是整个ICD-10双重分类系统的经典教学案例。AD的神经病理改变（β-淀粉样蛋白沉积、神经原纤维缠结）属于G30（神经系统变性病），由此导致的认知功能损害属于F00-F03（器质性精神障碍）。在ICD-10中，G30为剑号（†）编码，F00-F03为对应的星号（*）编码。

5.2 完整的AD星剑号对应表

剑号（病因）星号（临床表现）临床特征发病年龄
G30.0† F00.0* 早发性AD痴呆 < 65岁
G30.1† F00.1* 晚发性AD痴呆 ≥ 65岁
G30.8† F00.2* 非典型或混合型AD痴呆 不限
G30.9† F00.9* 未特指的AD痴呆 不限

5.3 编码流程

主诊断（首页第1行）：G30.0† 早发性阿尔茨海默病 —— 剑号码，驱动DRG入组
其他诊断（首页第2行）：F00.0* 早发性阿尔茨海默病性痴呆 —— 星号码，记录临床表型

注意：两个码都必须填报，且G30.0†必须放在主诊断位置

5.4 与F01/F03的鉴别——最重要的一组"不包括"规则

编码疾病核心特征与AD的关系
F01.x 血管性痴呆 阶梯样恶化 + 局灶神经体征 + 影像有脑血管病证据 与G30的"不包括"互斥——血管性痴呆不能用G30编码
F02.0* 皮克病性痴呆 额颞叶萎缩，早期人格改变>记忆减退 G31.0† + F02.0*，不同于AD的双码路径
F02.3* 帕金森病性痴呆 帕金森运动症状出现后 > 1年出现痴呆 G20† + F02.3* 或 G31.8† + F02.3*（DLB）
F03 未特指的痴呆 无法确定病因的痴呆综合征 不得与G30联用——如果病因明确为AD，必须用F00*而非F03

🧬 六、三大重要疾病编码详细拆解

6.1 多发性硬化（G35）——脱髓鞘的"旗舰病"

多发性硬化（MS）是G35的唯一疾病，没有亚目细分。但它与多个类似疾病有精细的鉴别编码：

临床表现编码鉴别要点
复发-缓解型MS（最常见类型） G35 时间多发 + 空间多发（McDonald标准）
视神经脊髓炎谱系疾病（NMOSD） G36.0 AQP4抗体阳性 + 长节段脊髓炎 ≥3椎体
急性播散性脑脊髓炎（ADEM） G04.0 单相病程，感染/疫苗接种后，多见于儿童
同心圆性硬化（Baló病） G37.5 MRI特征性层状同心圆病灶
进行性多灶性白质脑病（PML） A81.2 注意——PML是JC病毒感染，归入A章而非G章

6.2 偏瘫（G81）——最容易"偷懒"的编码

偏瘫编码G81.x是编码员最常看到的神经科诊断之一，但它的使用有严格限制：

⚠️ G81偏瘫的"不包括"清单（卷一核心注释）：
  • 先天性脑瘫（G80）——从出生就有的，不能用G81
  • 脑梗死所致偏瘫，急性期 → I63 + G81（偏瘫作为附加码）
  • 脑出血所致偏瘫，急性期 → I61 + G81
  • 中风后遗症期（发病 > 1年，病情稳定）→ I69.x，不使用G81
  • 暂时性肢体无力（如TIA后）→ G45，不用G81

核心逻辑：G81偏瘫是症状层面的编码，描述的是"哪里瘫"而不是"为什么瘫"。如果导致偏瘫的病因已知且处于活动期，应同时编码病因+偏瘫；如果属于后遗症期，则用I69的中风后遗症编码替代。

6.3 吉兰-巴雷综合征（G61.0）——快速进展的周围神经病

吉兰-巴雷综合征（GBS）是编码员容易混淆的疾病——它在临床上常表现为"四肢无力"，容易被误编入G82（截瘫/四肢瘫）或G70（重症肌无力）。

特征吉兰-巴雷 G61.0重症肌无力 G70.0脊髓病变 G82
起病方式 急性/亚急性（数天-4周） 波动性（晨轻暮重） 急性或慢性
受累分布 对称性，从下肢向上发展 眼肌 > 延髓肌 > 四肢 水平面以下
腱反射 减弱或消失 正常 亢进或减弱（取决于节段）
关键检查 脑脊液蛋白-细胞分离 + 神经传导速度减慢 AChR抗体 + 重复电刺激递减 脊柱MRI
前驱事件 常有（感染/疫苗/手术） 无 无特征性
💡 编码要点：GBS如果继发于特定感染（如空肠弯曲菌肠炎），应同时编码：感染病原体编码（作为其他诊断）+ G61.0（主诊断）。但注意不要使用A04.5（细菌性肠炎）作为主诊断——因为本次住院的核心问题是GBS，不是肠炎。

💡 七、重难点深度解析：G47 vs F51 —— 睡眠障碍的器质性/非器质性分水岭

睡眠障碍编码是编码考试和质控检查的"终极杀手题"。为什么难？因为同一个"失眠"诊断，在ICD-10中根据有无器质性病因分别归入G47和F51两个不同的章。

7.1 两条路径的完整对比

维度G47 器质性睡眠障碍F51 非器质性睡眠障碍
归属 G章（神经系统疾病） F章（精神和行为障碍）
核心特征 由明确的器质性病因（神经系统/躯体疾病）引起 心理社会因素为主，无已知器质性病因
典型场景 发作性睡病G47.4、睡眠呼吸暂停G47.3、周期性肢体运动障碍G47.6 非器质性失眠F51.0、梦魇F51.5、睡行症（梦游）F51.3
DRG入组 B类（神经系统疾病） C类（精神疾病）——入组完全不同
是否需要多导睡眠图 是——PSG是诊断的金标准 否——以临床访谈和量表为主

7.2 G47各亚目的编码细则

亚目疾病诊断要点常见编码陷阱
G47.0 入睡和维持睡眠障碍（器质性失眠） 由帕金森病、脑卒中、慢性疼痛等器质性疾病直接导致的失眠 必须要有明确的原发病诊断支撑
G47.1 过度嗜睡（器质性） 发作性睡病除外——发作性睡病专门归入G47.4 甲状腺功能减退、肥胖低通气等导致的日间嗜睡
G47.2 睡眠-觉醒周期障碍 昼夜节律紊乱，如轮班工作睡眠障碍、睡眠时相延迟综合征 非器质性的昼夜节律障碍用F51.2
G47.3 睡眠呼吸暂停 阻塞性（OSA）或中枢性，PSG证实AHI ≥5 极高频编码——合并高血压、冠心病时需分别编码
G47.4 发作性睡病和猝倒症 四联征：日间嗜睡+猝倒+睡眠麻痹+入睡前幻觉 不要与G47.1的过度嗜睡混淆——有猝倒才归入G47.4

7.3 神器：G47/F51分水岭三步决策树

患者以"失眠/睡眠障碍"就医 │ ├─【第一步】是否已做多导睡眠图（PSG）？ │ NO → 无客观证据支撑器质性诊断 → 倾向 F51 │ YES → 进入第二步 │ ├─【第二步】是否有明确器质性病因？ │ ├─ 帕金森病、脑卒中、慢性疼痛 → G47.0 │ ├─ 睡眠呼吸暂停（AHI≥5）→ G47.3 │ ├─ 周期性肢体运动（PLMI≥15）→ G47.6 │ ├─ 有猝倒发作 → G47.4（发作性睡病） │ └─ 无上述器质性发现 → 进入第三步 │ └─【第三步】心理社会因素是否为主要驱动？ ├─ 焦虑/抑郁导致的失眠 → F51.0（非器质性失眠） ├─ 梦魇 → F51.5 ├─ 睡行症（梦游）→ F51.3 └─ 无法判定器质性/非器质性 → F51.9（未特指） 但需注意：此选严重影响DRG入组，应尽量明确归类
⚠️ 终极陷阱：如果患者的失眠同时存在器质性病因（如帕金森病）和心理因素（如焦虑），编码员应遵循"病因为主"原则，将失眠编入G47.0（器质性），将焦虑编为其他诊断。不能因为合并焦虑就改用F51.0，因为"有器质性病因"这一条就足以使失眠脱离F章。

1
基础题 · G章码位归属判断 PRACTICE

请判断以下疾病分别属于G章的哪个板块，并给出对应的码位范围：

  1. 多发性硬化
  2. 亨廷顿病
  3. 吉兰-巴雷综合征
  4. 亨特综合征（Ramsay Hunt 综合征，膝状神经节带状疱疹）
点击查看答案

✅ 答案：

1. 多发性硬化 → G35-G37（脱髓鞘疾病），编码 G35

2. 亨廷顿病 → G10-G14（系统性萎缩），编码 G10

3. 吉兰-巴雷综合征 → G60-G64（多神经病和周围神经），编码 G61.0

4. 亨特综合征 → 注意陷阱！虽然面神经麻痹是神经科表现，但病因是水痘-带状疱疹病毒（VZV）。ICD-10编码 B02.2†（带状疱疹累及其他神经系统）+ G53.0*（带状疱疹后膝状神经节炎）。G53为星号（*），表示分类于他处的疾病引起的神经病变。该病不属于G章板块⑦（G50-G59），而是通过带状疱疹编码表达病因。

2
基础题 · 癫痫分型编码 PRACTICE

某患者因癫痫发作入院。脑电图显示双侧同步3Hz棘慢波，临床表现为短暂意识丧失（5-10秒），无抽搐，发作后立即清醒。

问题：应选择哪个编码？请说明理由，并解释为什么不能用G40.3。

点击查看答案

✅ 正确答案：G40.3（全身性特发性癫痫）—— 实际上是可以用G40.3的！

推理过程：

①脑电图双侧同步3Hz棘慢波 → 典型失神发作的EEG特征

②短暂意识丧失、无抽搐、立即清醒 → 失神发作（Absence Seizure），属于全身性发作

③失神发作是特发性全身性癫痫的一种类型，编码为 G40.3

💡 补充说明：

如果EEG显示一侧脑区异常放电且意识保留 → G40.0/G40.1/G40.2

如果发作持续 > 15分钟或连续发作间期意识未恢复 → G41.1（失神持续状态）

本题的关键在于：双侧同步放电 = 全身性 = 用G40.3，而非G40.2。

3
基础题 · TIA vs 脑梗死鉴别 PRACTICE

患者72岁男性，因"突发左侧肢体无力2小时"入院。急诊CT未发现脑出血。入院后MRI-DWI提示右侧放射冠区高信号（急性梗死灶）。神经内科医生在出院小结中写下诊断"TIA（短暂性脑缺血发作），腔隙性脑梗死"。

问题：作为编码员，你应该如何处理这个诊断？主诊断应编什么？

点击查看答案

✅ 第一步·判断：

医生写的是"TIA"+"腔隙性脑梗死"，但MRI-DWI已证实存在急性梗死灶——这就不再是TIA。TIA的定义性特征之一就是DWI阴性。

✅ 第二步·术语清洗：

"TIA"应当被剔除——因为DWI阳性已经推翻了TIA的诊断。"腔隙性脑梗死"才是本患者的实质诊断。

✅ 第三步·编码：

主诊断：I63.801（腔隙性脑梗死）

不能编码为G45.x（TIA系列）

💡 核心教训：DWI有高信号 = 脑梗死 ≠ TIA。这是DRG入组的分水岭——I63和G45的权重差异可能超过2倍。医生即使写了TIA，编码员也必须以影像学客观证据为准。

4
进阶题 · 帕金森鉴别与星剑号 CHALLENGE

患者78岁男性，入院诊断为"帕金森病"。住院期间神经心理评估发现：MMSE 18分（认知功能明显下降），伴有视幻觉（报告"看到有小孩在病房里"），认知障碍出现时间与帕金森运动症状间隔约2年。进一步检查未提示血管性病因。

问题：请给出完整的疾病编码（包括所有诊断和星剑号配对），并说明G20 vs G31.8†的判断依据。

点击查看答案

✅ 诊断分析：

关键是"1年规则"：帕金森症状先于痴呆出现 > 1年 → 帕金森病性痴呆（PDD）

帕金森症状与痴呆同时或痴呆在先（差值 < 1年）→ 路易体痴呆（DLB）

本题：帕金森运动症状在先，间隔约2年才出现认知障碍 → 满足">1年"条件 → PDD

✅ 完整编码：

①主诊断：G20† 帕金森病（原发性）

②其他诊断：F02.3* 帕金森病性痴呆

推理要点：

①发病顺序是关键：帕金森→痴呆（差值>1年）= PDD，用G20† + F02.3*

②痴呆+帕金森几乎同时出现（差值<1年）= DLB，用G31.8† + F02.3*

③视幻觉在DLB中更核心，但PDD患者也可见——不能单凭视幻觉就判定为DLB

④本题核心决策依据是运动症状与认知症状的发作时间间隔 = 2年

💡 如果误编：编成G31.8†会导致与临床事实不一致，且DRG入组路径差异大。

5
进阶题 · 睡眠障碍 G47 vs F51 分水岭 CHALLENGE

患者因"严重失眠半年"入院。PSG检查显示：AHI = 28次/小时（重度阻塞性呼吸暂停），同时存在中度焦虑（HAMA 16分）。患者自述入睡困难、夜间多次醒来。

问题：失眠应编入什么？给出完整的诊断编码列表，并解释每条编码的排序逻辑。

点击查看答案

✅ 完整编码方案：

①主诊断：G47.3（睡眠呼吸暂停——OSA），AHI=28证实为重度

②其他诊断1：F41.1（广泛性焦虑障碍，中度）

③其他诊断2：G47.0（器质性失眠——由OSA直接导致）

✅ 排序逻辑（"病因为主"原则）：

① G47.3 做主诊断——因为OSA是本次住院发现的核心器质性病因，且AHI=28属于中度以上，有治疗干预必要

② 失眠是由于 OSA 导致的反复夜间低氧和觉醒引起的，满足"器质性病因→失眠"的因果关系 → 用 G47.0 而非 F51.0

③ 焦虑虽然存在，但它是次要因素——不能因为有焦虑就把失眠编入F51.0

💡 决策逻辑链：PSG有客观异常（AHI↑） → 失眠由器质性因素引起 → G47.0 ✓（不是F51） → OSA是根本病因 → G47.3 做主诊断 ✓

6
进阶题 · 多诊断冲突的综合编码 CHALLENGE

患者65岁，因"右侧肢体无力伴言语含糊3天"入院。住院期间记录以下发现：

  • 头颅MRI-DWI：左侧基底节区及放射冠多发急性腔隙性梗死
  • 神经系统检查：右侧肢体肌力Ⅲ级，右侧病理征阳性，构音障碍
  • 既往于2年前曾患"左侧基底节脑梗死"，遗留轻度左手不灵活（已稳定1年以上）
  • 既往有高血压病史10年，口服降压药控制尚可

问题：请列出本次住院的完整编码（主诊断 + 所有其他诊断），并解释2年前的脑梗死后遗症如何处理。

点击查看答案

✅ 完整编码方案：

主诊断：I63.801（腔隙性脑梗死——本次新发的急性梗死）

其他诊断：

② G81.9（偏瘫，右侧——急性梗死导致的新偏瘫）

③ I69.301（脑梗死后遗症，左侧肢体轻瘫——2年前的陈旧性后遗症）

④ I10（原发性高血压——慢性基础病）

✅ 关键决策解析：

① 2年前的脑梗死后遗症（左手不灵活）：

——发病已 > 1年，病情已稳定 → 应使用I69.301（脑梗死后遗症，单侧上肢轻瘫）

——不能使用G81.x或I63.x——因为急性期已过，且病情已稳定

——如果本次入院只是后遗症存在而没有新发梗死，I69.301就是主诊断

② 本次新发偏瘫（右侧肢体肌力Ⅲ级）：

——由急性腔隙性脑梗死引起，属于活动期疾病 → 同时编I63.801 + G81.9

——注意区别：新发急性偏瘫（I63+G81）vs 陈旧后遗症（I69），时机是关键

③ 高血压：

——作为脑梗死的危险因素，作为其他诊断I10

💡 高阶认知：同一患者身上同时存在"新发急性偏瘫"（I63+G81）和"陈旧性后遗症"（I69），这在神经内科中极为常见——二者不能混为一谈，必须分别编码。
