📖 一、术语清洗是什么？——不只是"把诊断写好看"

1.1 定义与本质

术语清洗（Terminology Cleaning）是指将临床医生书写的原始诊断（口语化、简写、含歧义的表达），转化为 ICD-10 认可的、可用于索引查找的标准诊断名称的过程。它是编码的第一道工序，也是决定最终编码质量的核心环节。

如果把编码过程比作做菜，术语清洗就是"洗菜、择菜"——菜没洗干净，后面刀工再好、火候再准，端上桌的也是脏菜。很多编码员的错误不在索引查错，而在第一步就没有把诊断"洗干净"。

术语清洗要回答三个问题：
① 这是什么病？（疾病本质——确定主导词方向）
② 这个诊断是否完整？（缺不缺部位/性质/病因等关键信息）
③ 这个诊断是否准确？（有没有把两个不同的病混为一谈）

1.2 为什么要做术语清洗——不洗的后果

场景临床写法不清洗直接编清洗后正确编后果差异
简写模糊"心梗"I21.9（未特指）I21.0（STEMI）或 I21.4（NSTEMI）I21.9 不能入 DRG F 组，权重损失约 0.3-0.5
诊断不完整"脑出血"I61.9（脑内出血，未特指）I61.001（壳核出血）质控扣分，统计不准
概念混淆"高血压性心脏病"I10（高血压）+ I51.9（心脏病）I11.9（高血压性心脏病）两个码变一个合并码，DRG 入组完全不同
症状当诊断"头晕"（入院出院都写）R42（头晕）I63.801（腔隙性脑梗死）R 码入组权重极低，医院损失大
⚠️ 真实数据：根据某三甲医院 2024 年的质控抽查，因术语清洗不到位导致的编码错误占比高达 37.8%，超过了索引查找错误（29.1%）和核对疏漏（22.5%）。术语清洗，不是可选项，是必答题。

🔬 二、术语清洗五步法——从模糊到精准的完整路径

1

识别歧义——"这句话可能有几种意思？"

拿到临床诊断后，先问自己：这个诊断是否只有一种解读？如果有两种以上的可能，就必须做下一步。

高歧义信号：简写（"心梗""功血"、"支扩"）、省略部位（"骨折"没写具体位置）、笼统词（"心脏病""肝病"）、外文缩写（"MI""COPD""TIA"）

例："脑出血" → 可能是 I60.x（蛛网膜下腔）/ I61.x（脑实质）/ I62.x（硬膜下）→ 必须查影像报告
2

查源补全——"哪些信息还没写？"

对照 ICD-10 对某个疾病的标准描述要素，回看病程记录、影像报告、化验单，把缺失的关键信息补上。

部位左侧/右侧、上叶/下叶、近端/远端？
性质急性/慢性/亚急性、化脓性/非化脓性？
病因细菌性/病毒性/缺血性/创伤性？
分期活动期/缓解期、早/中/晚期、Ⅰ/Ⅱ/Ⅲ/Ⅳ级？
3

标准命名——"ICD-10 怎么叫这个病？"

将临床用语翻译成 ICD-10 标准命名。这一步需要熟悉 ICD-10 的术语习惯。

翻译规则：ICD-10 偏好"解剖部位 + 病理改变"的双段命名，而非症状式命名。

临床："胃疼，反酸水"
→ 标准：慢性胃炎（K29.5）或 胃食管反流（K21.9）
                   ← 看胃镜结果决定
4

验证一致性——"诊断和编码对得上吗？"

清洗后的标准诊断要与病程记录、检查结果相互印证。特别注意：

  • 影像报告中的描述是否与清洗后的诊断一致？
  • 化验单是否符合该诊断的典型特征？
  • 手术记录中的术中所见是否印证了诊断？

如果发现矛盾——优先相信客观证据（病理、影像），而不是医生的初诊描述。

5

输出档——"写成能进索引的标准表达"

最终输出的标准诊断名称要满足三个条件：

  • 唯一性——只有一个 ICD-10 编码与之对应
  • 完整性——包含索引查找所需的所有修饰词
  • 可索引性——主导词明确，能在第三卷索引中找到
清洗完成例：
临床：肝不好，小三阳
  → 标准：慢性乙型病毒性肝炎，HBeAg阴性
  → 编码：B18.100x003

🔄 三、六类常见诊断陷阱——逐一击破

3.1 陷阱一：简写与俗称

临床医生习惯用简称，这些简称在科室内部大家都懂，但进到编码系统里就变成了"薛定谔的诊断"。

临床简写/俗称可能的含义（≥2种）需要查什么来确定最终标准诊断方向
"心梗"急性STEMI / 急性NSTEMI / 陈旧性心梗心电图、心肌酶（CK-MB、cTnI）I21.0-I21.4 / I25.2
"功血"功能性子宫出血 → 排卵障碍性异常子宫出血妇科超声、激素六项N93.8（ICD-10 名：子宫出血，异常的，其他特指的）
"支扩"支气管扩张症 / 支气管扩张伴感染痰培养、血象J47 / J47+J98.4（视有无感染）
"甲亢"Graves病 / 毒性结节性甲状腺肿 / 亚急性甲状腺炎甲亢期甲状腺超声、TRAb、吸碘率E05.0 / E05.2 / E06.1
"COPD"慢性阻塞性肺病稳定性 / 急性加重期血气分析、肺功能J44.9 / J44.1（急性加重）
"脑梗"急性脑梗死 / 腔隙性脑梗死 / 陈旧性脑梗死CT/MR+DWI、发病时间I63.x / I63.801 / I69.3

❌ 错误做法

看到"心梗"就直接编 I21.9（未特指），因为觉得"反正是心梗，差不到哪去"

→ 结果：I21.9 权重远低于 I21.0-4，DRG 入组损失严重

✅ 正确做法

回看病程记录和心电图报告，确定 STEMI 还是 NSTEMI，再定位到具体部位（前壁/下壁/后壁等）

→ 结果：准确编码到 I21.0-I21.4 的亚目层级，DRG 入组正确

3.2 陷阱二：缺部位/侧别的诊断

很多疾病的 ICD-10 编码直接依赖解剖部位，不补全部位等于自废武功。

例："肺癌"

查病理报告 → 确定肺叶（上叶/中叶/下叶）和病理类型（鳞癌/腺癌/小细胞癌）

→ 上叶支气管肺癌，鳞状细胞癌：C34.101 M8070/3

例："股骨骨折"

查影像报告 → 确定具体部位（股骨头/股骨颈/股骨干/转子间）和类型（闭合/开放）

→ 股骨颈骨折，闭合性：S72.002

例："肾炎"

查肾活检（如有）→ 确定IgA肾病/膜性肾病/微小病变/狼疮性肾炎等

→ IgA肾病：N02.801

例："脑出血"

查头颅CT → 确定出血部位（壳核/丘脑/脑叶/小脑/脑干）和类型

→ 壳核出血：I61.001；蛛网膜下腔出血：I60.x

⚠️ 核心原则：诊断没有部位 → 编码一定不精确 → 质控一定扣分。不要替医生猜部位——没有写就去查病历，病历也没有就去问医生。编码员的底线是"不编没证据的码"。

3.3 陷阱三：症状当诊断（R码滥用）

这是质控检查中扣分率最高的错误类型之一。第十八章（R00-R99）是"症状、体征和异常临床所见"，其使用有严格限制。

规则说明示例
有确诊 → 不用症状 住院期间明确了器质性病因的，必须用病因码替代症状码 "头晕"入院 → 确诊脑梗死 → 主诊断改为 I63.x，不可再用 R42
始终未确诊 → 可用症状 出院时仍不能明确病因的，可以使用症状码作为主诊断 "发热待查"入院 → 各种检查均阴性 → 出院诊断可以是 R50.9
不再进一步找病因的既知病 → 不用症状 如果某症状是已知疾病的典型表现，不能用症状码替代该疾病 已知冠心病患者"胸痛"入院 → 应编冠心病（I25.1），非胸痛（R07.4）
口诀：病因明确换病因，始终不明留症状；已知病因复发作，不能偷懒用症状。

3.4 陷阱四：笼统诊断——"XX病""XX炎"

临床医生有时用很笼统的诊断名称，例如"肝病""心脏病""肺炎"。这些诊断没有提供足够的信息来确定编码。

笼统诊断要追问的信息可能的编码方向
"肝病"病毒性/酒精性/药物性/脂肪性？急性/慢性？有肝硬化吗？B18.x / K70.x / K71.x / K76.0 / K74.x
"心脏病"冠心病/扩心病/肥厚性心肌病/瓣膜病/心律失常？I20-I25 / I42.0 / I42.1-I42.2 / I34-I37 / I47-I49
"肺炎"大叶性/支气管性/间质性？病原体是什么？J12-J18 中根据病原体细分
"贫血"缺铁性/巨幼细胞性/溶血性/再生障碍性？D50.x / D51-D53 / D55-D59 / D60-D61
"肾功能不全"急性/慢性？分期（1-5期）？透析？N17.x / N18.x / Z99.2

3.5 陷阱五：外文缩写与混合用语

病历中大量使用英文缩写（MI、CABG、PCI、TIA、ERCP、PTCA等）以及中英文混合诊断。这些缩写看似标准，但在不同科室可能含义不同。

缩写全称标准中文诊断常见编码注意事项
MIMyocardial Infarction急性心肌梗死I21.x需区分 STEMI / NSTEMI
CABGCoronary Artery Bypass Grafting冠状动脉旁路移植术36.1x（手术码）是手术操作，不是疾病诊断
TIATransient Ischemic Attack短暂性大脑缺血发作G45.x与脑梗死区分：TIA 24h内恢复，脑梗死有影像证据
ERCPEndoscopic Retrograde Cholangiopancreatography内镜逆行胰胆管造影51.10 / 51.87诊断性 vs 治疗性（取石）编码不同
CAPCommunity-Acquired Pneumonia社区获得性肺炎J15.9 或细分有病原体时改用具体病原体编码
HAPHospital-Acquired Pneumonia医院获得性肺炎J15.9 + Y95注意附加编码外因 Y95

3.6 陷阱六：易混淆的疾病对——一字之差，编码千里

有些疾病在临床用语上非常接近，但在 ICD-10 中归入了完全不同的类目甚至章节。术语清洗时必须特别警惕。

易混淆对关键区别A的编码B的编码
脑梗死 vs 脑出血缺血性 vs 出血性（看CT有无高密度灶）I63.xI61.x
房颤 vs 房扑心律绝对不齐 vs 规则或不规则I48.xI48.x（同亚目但有细分）
肝硬化 vs 肝炎终末纤维化 vs 活动性炎症K74.xB15-B19 / K73.x
骨关节炎 vs 类风湿性关节炎退行性 vs 自身免疫性M15-M19M05-M06
2型糖尿病 vs 1型糖尿病胰岛素抵抗为主 vs 胰岛素绝对缺乏E11.xE10.x
COPD vs 哮喘不可逆气流受限 vs 可逆性气道阻塞J44.xJ45.x

🧠 四、重难点深度解析：模糊诊断的边界——"该放弃症状码吗？"

这是术语清洗中最让新手困惑的场景：临床诊断看起来很明确，但 ICD-10 的分类轴心和临床诊断的分类轴心不一样。这时候怎么洗？

4.1 经典冲突场景："急性左心衰"

深度分析："急性左心衰"的术语清洗

背景：患者因"呼吸困难、端坐呼吸"入院，胸片提示肺水肿，超声提示左心室射血分数降低（LVEF 35%）。医生出院诊断："急性左心衰竭"。

问题：急性左心衰是一种临床表现（I50.1），但它背后有病因——是什么导致了心衰？ICD-10 的规则是：如果心衰有明确病因，应优先编码病因，心衰作为附加诊断。

如果有…主诊断附加诊断逻辑
高血压多年，心脏扩大I11.0（高血压性心脏病伴心衰）I50.1（左心衰）合并编码 → 高血压病因驱动
冠心病+前壁心梗病史I25.5（缺血性心肌病）I50.1缺血性心肌病 → 病因驱动
风湿性二尖瓣狭窄I05.0（风湿性二尖瓣狭窄）I50.1瓣膜病 → 病因驱动
找不到明确病因I50.1（左心衰竭）—无病因 → 心衰做主诊断

术语清洗的核心动作：不能只看"急性左心衰"这四个字。要像侦探一样追问：为什么心衰？追到病因才算洗干净。

❌ 仅洗表面

急性左心衰 → I50.1
（没有追问病因）

漏了病因编码，DRG 入组偏差，医院损失

✅ 深挖病因

高血压性心脏病伴心衰 → I11.0
+ I50.1 左心衰

病因+表现双码完整，DRG 入组准确

4.2 过度清洗的警惕——不要"美化"诊断

术语清洗也有边界。不是诊断越"高级"越好。以下情况不能清洗：

过度清洗行为为什么不对正确做法
把"胸痛待查"洗成"冠心病"检查结果没支持冠心病诊断已确诊→用病因；未确诊→保留 R07.4
把"腹部不适"洗成"慢性胃炎"胃镜没做，不能推定用R10.4（其他腹部未特指疼痛），除非胃肠镜确诊
把"II型糖尿病"洗成"II型糖尿病伴肾病"尿蛋白正常，无肾病证据仅编 E11.9，不要"升级"并发症
⚠️ 术语清洗的底线：清洗是"澄清"和"标准化"，不是"脑补"和"升级"。所有清洗后的诊断必须有客观证据支撑（化验单、影像报告、病理报告）。没有证据的"清洗"叫编造——是编码员的核心红线。

✏️ 五、今日案例练习（共 6 题）

1 简写诊断清洗 基础

临床医生写了以下简写诊断。请说明每个诊断需要补全哪些信息才能开始编码：

  1. "支扩"（支气管扩张）
  2. "T2DM"（2型糖尿病）
  3. "PUD"（消化性溃疡）
点击查看答案

1. "支扩"：需明确是否伴有急性感染（看血象、CRP、痰培养）→ 单纯支扩 J47 / 支扩伴感染 J47+J98.4

2. "T2DM"：2型糖尿病本身 E11.9。但关键要看有没有并发症——肾病（尿蛋白）、眼病（眼底检查）、神经病变（肌电图）、周围循环（血管超声）。有并发症要用合并编码 E11.2-E11.6。另外"T2DM"这个缩写本身要认出是 Type 2 Diabetes Mellitus。

3. "PUD"（Peptic Ulcer Disease）：需明确：① 部位（胃/十二指肠/吻合口）② 急性还是慢性 ③ 有无出血/穿孔/梗阻 → 编码从 K25.x-K28.x 中根据具体情况选择。

总结：三个简写的共同点——都只说了"病名"，都把"修饰词"省略了。而 ICD-10 编码恰恰依赖修饰词来区分亚目。

2 缺部位诊断补全 基础

以下诊断缺少关键部位信息。请说明应该查阅什么资料，并给出可能的编码范围：

  1. "下肢骨折" → 查什么？编码范围？
  2. "颅内出血" → 查什么？编码范围？
  3. "消化系统肿瘤" → 查什么？编码范围？
点击查看答案

1. "下肢骨折"：查 X光片/CT报告 → 具体骨（股骨/胫骨/腓骨/足骨…）+ 具体部位（近端/颈/干/踝…）+ 闭合/开放 → 编码范围 S72-S92（因部位差异极大）

2. "颅内出血"：查头颅CT/MR → 明确是蛛网膜下腔（I60）、脑实质（I61）、硬膜下（I62.0）、硬膜外（S06.4）还是硬膜下非创伤性（I62.0）。这四者编码和 DRG 入组完全不同。

3. "消化系统肿瘤"：查胃镜/肠镜+病理报告 → 具体部位（食管/胃/十二指肠/结肠/直肠…）+ 病理类型 + 良恶性 → 编码范围 C15-C26 中是具体器官编码。没有病理就不能确定良恶性。

关键原则：看到笼统部位诊断 → 立刻打开影像/内镜/病理报告补全。

3 症状码→病因码转换 基础

判断以下场景中，出院主诊断应该用症状码还是病因码，并说明理由：

场景入院诊断住院检查结果你的主诊断选择
A腹痛腹部CT + 结肠镜确诊乙状结肠腺癌？
B发热待查所有检查均阴性，发热原因不明？
C头痛头颅MRI确诊脑膜瘤？
点击查看答案

A：主诊断应为 C18.7（乙状结肠恶性肿瘤）。确诊了恶性肿瘤，必须用恶性肿瘤编码，不能用 R10.x（腹痛）。

B：主诊断应为 R50.9（发热，未特指）。住院期间未明确病因 → 可以且应该使用症状码做主诊断。

C：主诊断应为 D32.0（脑膜瘤，良性）。确诊了肿瘤（即使是良性的），也必须用肿瘤编码替代症状码 R51（头痛）。

判断口诀：住院有确诊 → 用确诊码；住院没确诊 → 用症状码。这个"判断"本身就是术语清洗的核心环节之一。

4 易混淆疾病术语清洗 进阶

某患者入院诊断为"脑卒中"，头颅CT提示左侧基底节区低密度灶（符合缺血性改变），既往有高血压病史10年。请完成完整的术语清洗并给出编码。

点击查看答案

术语清洗过程：

① "脑卒中"→ 缺血还是出血？CT报告显示"低密度灶"= 缺血性改变，排除出血 → 方向从 I60-I62（出血）转向 I63（脑梗死）。

② 部位？基底节区 ≠ 单一结构。需要进一步查 CT/MR 精确到壳核 or 丘脑 or 尾状核 → 假设为壳核 → I63.5 大类。

③ 病因分析：有高血压病史10年 → 高血压是导致脑梗死的危险因素，但 ICD-10 的脑梗死编码（I63）是按责任血管区分的（大脑前/中/后动脉、椎-基底动脉等），而非按病因。因此高血压单独编码 I10，不与 I63 合并。

最终编码：主诊断 I63.501（大脑中动脉闭塞引起的脑梗死，缺血性卒中）或 I63.801（腔隙性脑梗死）依具体影像报告而定；附加诊断 I10（高血压病）。

核心心得：脑卒中（stroke）是"伞"——包含脑梗死（ischemic）和脑出血（hemorrhagic）。术语清洗的第一步就是把伞打开，看清下面到底是谁。

5 合并编码的术语清洗判断 进阶

某患者出院小结列出三个诊断：

  1. 高血压病
  2. 慢性心力衰竭
  3. 慢性肾脏病3期（eGFR 45）

病程记录显示：高血压病史15年，心脏彩超提示左心室肥厚、舒张功能减退。请判断：
(a) 是否应该使用合并编码？
(b) 最终的编码方案是什么？

点击查看答案

(a) 是否使用合并编码？

第一步，检查高血压与心脏病的关系：病程记录明确显示高血压病史15年 + 心脏彩超左心室肥厚 → 存在因果关系 → 应使用合并编码 I11.0（高血压性心脏病伴心力衰竭），而非 I10 + I50.9 分开编。

第二步，检查高血压与肾病的关系：患者同时有 CKD3期（N18.3）和高血压 → ICD-10 提供了 I12.0 和 I13.x。如果肾病也是高血压导致的（高血压性肾病），则需用 I13.x（同时有心肾合并）。但此处肾病原因是否高血压导致的需进一步确认——假设为高血压性肾病。

(b) 最终编码方案：

如果有证据支持高血压同时导致心脏病和肾病 → I13.0 高血压性心肾衰竭（一个码封装三者！）

如果肾病原因不明确或非高血压性 → 主诊断 I11.0 高血压性心脏病伴心衰 + 附加 N18.3 CKD3期

核心心得：这是一个"术语清洗 + 合并编码判断"的组合拳。先把"高血压""心衰""肾衰"三个诊断之间的关系理清楚，再决定是分开编还是合并编。

6 综合术语清洗实战 进阶

病例：

患者，男，72岁。入院诊断："COPD急性加重；肺气肿；吸烟史40年"。住院期间行血气分析（PaO₂ 58mmHg，PaCO₂ 55mmHg），肺功能（FEV1/FVC < 0.7，FEV1 35%预计值），痰培养（肺炎克雷伯菌），出院诊断医生写："慢阻肺急性加重；慢阻肺伴下呼吸道感染；肺气肿"。

问题：

  1. 医生的三个诊断中有没有重复的？术语怎么清洗？
  2. 最终的主诊断和其他诊断各是什么？（请给出编码）
  3. 还需不需要编码吸烟史？
点击查看答案

1. 术语清洗：

① "COPD急性加重"和"慢阻肺急性加重"是同一个意思 → 去重，留"慢性阻塞性肺疾病伴急性加重"。
② "肺气肿"是 COPD 的组成部分——COPD 本身就包括慢性支气管炎和/或肺气肿。如果该患者的 COPD 以肺气肿为主要表现（肺功能+影像支持），则不需要单独编码肺气肿。但如果医生明确要强调肺气肿，可在 COPD 的诊断描述中体现，编码仍归 J44.x。
③ "慢阻肺伴下呼吸道感染"——根据痰培养结果为肺炎克雷伯菌，这是确定的细菌感染。ICD-10 有 J44.0 慢性阻塞性肺病伴急性下呼吸道感染。

2. 最终编码：

主诊断：J44.0（慢性阻塞性肺病伴急性下呼吸道感染）——这个码同时表达了COPD + 急性感染两个核心问题。
附加诊断：J96.1（慢性呼吸衰竭，PaO₂ <60mmHg，PaCO₂ >50mmHg 支持慢性呼衰）
【不需要单独编码肺气肿 J43.9，因为 J44.0 已经包含了 COPD 的病理范围】

3. 吸烟史是否需要编码？

如果吸烟史对本次诊疗有影响（如出院医嘱强调戒烟）→ 应编码 Z72.0（烟草使用）。在 DRG/DIP 时代，这个 Z 码虽然不驱动入组，但反映诊疗的完整性。有些地区质控要求对有吸烟史的 COPD 患者必须编码 Z72.0。

综合心得：这个案例展示了术语清洗的三个层次——去重（三合一）、归类（肺气肿归入 COPD）、精确定位（急性感染 → J44.0）。每一步都要依据证据，而非凭感觉。

✅ 六、今日重点回顾

1 术语清洗是编码第一道工序，不洗就编=地基不牢
2 五步法：识别歧义→查源补全→标准命名→验证一致性→输出档
3 六大陷阱：简写、缺部位、症状当诊断、笼统诊断、外文缩写、易混淆对
4 病因明确→用病因码；始终未确诊→保留症状码；绝不可把模糊诊断"升级"
5 心衰、脑卒中、肺炎等"伞状诊断"必须追问病因/分型
6 清洗的底线：有证据才能洗，无证据不能编——影像/化验/病理是铁证
7 "高血压+心衰+肾衰"三联要判断因果关系，决定是否用 I13.x 合并编码
8 Z72.0（吸烟史）等生活方式编码在 DRG 时代虽不驱动入组，但体现完整性
