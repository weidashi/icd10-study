📖 一、A-B章总论——特殊组合章的"边界感"

1.1 什么是"特殊组合章"？

在ICD-10的22个章节中，大多数章节按解剖部位（如循环系统I、呼吸系统J）或疾病性质（如肿瘤C/D、损伤S/T）分类。但第1章（A00-B99）是一个特殊组合章——它以病因为核心轴心，将"有传染性或可传播性"的疾病汇集在一起，不论其侵犯的是哪个解剖部位。

infection ≠ 传染病：ICD-10 英文原文中第1章标题为 "Certain infectious and parasitic diseases"，其中 infection 不仅指可传播的传染病，还包括局限性和不可传播的感染。中文翻译为"传染病"容易误导编码员——并不是所有感染都归入本章。

1.2 本章的核心"不包括"——哪些感染不在这里？

本章有五大类明确不包括的内容，这些是最容易"编错章"的情况：

不包括类别应编码至典型示例编码员的检查点
病原携带者/可疑携带者 Z22.- 乙肝表面抗原携带者 → Z22.5（注意：Z22.5已被删除，属ICD-10更新） 无临床症状、仅有实验室阳性 → 不是本章
非传染性病因的局部感染 各解剖章节 细菌性肺炎 → J15（呼吸系统）、急性肾盂肾炎 → N10（泌尿系统） 局限在一个器官/系统的感染 → 归入该器官章节
妊娠分娩产褥期感染 O98.- / O23.- 妊娠期泌尿生殖道感染 → O23；非泌尿生殖道感染 → O98.3 患者是孕妇 → 第15章优先
围生期感染 P35-P39 先天性梅毒 → A50（例外：按病因入本章）、新生儿腹泻 → P78.3 子宫内或分娩时获得 → 第16章
流感和急性呼吸道感染 J00-J22 流感 → J09-J11、急性上呼吸道感染 → J06 呼吸系统首选第10章

1.3 编码规则：默认急性

⚠️ 关键规则：临床未明确传染病或寄生虫病是慢性者，假定分类为活动性或急性。这是本章独有的编码假定规则。例如：出院诊断为"乙型病毒性肝炎"但未标明急慢性，默认分类为 B18.1（慢性乙型肝炎）——此为ICD-10更新后的规定。

🫁 二、结核病编码（A15-A19）——最易出错的分类轴心

2.1 分类结构

呼吸系统结核的核心分类轴心不是解剖部位，而是实验室对结核杆菌检查的证实情况。这是A15与A16的根本区分依据，也是编码员最容易忽略的"暗线"。

类目编码名称分类依据"不包括"
A15 经细菌学和组织学证实的呼吸道结核 痰涂片(+)、痰培养(+)、病理活检(+)、GeneXpert(+) 先天性结核(P37.0)
HIV造成的结核(B20.0)
结核性硅沉着病(J65)
与结核有关的肺尘埃沉着病(J65)
A16 未经细菌学或组织学证实的呼吸道结核 涂片(-)、培养(-)、或未提及检查结果
A17† 神经系统的结核 解剖部位 + 临床表现（星剑号编码）
A18 其他器官的结核 按解剖部位细分（骨结核、肾结核……）
A19 粟粒性结核 急性(A19.0单部位/A19.1多部位/A19.2) vs 慢性(A19.8)

2.2 A15 vs A16——编码员必须看化验单

❌ 错误做法

临床诊断写"继发性肺结核"，编码员不看化验结果就直接查索引：结核→肺→A16.2（未提及证实情况）

结果：漏掉A15，DRG入组可能不同

✅ 正确做法

第一步：看病程记录/化验报告，确认是否有痰涂片、痰培养、病理活检等结果

第二步：若细菌学或组织学证实 → A15；无检查结果或检查阴性 → A16

2.3 A15类目下的细化编码

亚目编码名称适用场景关键检查点
A15.0肺结核，经痰液镜检证实痰涂片抗酸杆菌(+)最常见的确诊形式，无论是否同时痰培养
A15.1肺结核，仅经痰培养证实痰涂片(-)，痰培养(+)诊断记录必须明确"仅培养阳性"
A15.2肺结核，经组织学证实经支气管镜/胸腔镜活检病理(+)需有病理报告支撑
A15.3肺结核，经（未特指）方法证实有阳性证据但方法未说明已证实但具体方法不清楚时使用
🗂️ 主导词：结核 → 肺 →（看证实情况分类）。粟粒性结核需区分急慢性：临床未明确时假定为急性；急性粟粒性结核还要区分单个部位(A19.0)或多部位(A19.1)。

2.4 结核病的"不包括"——三个特殊的交叉编码场景

  • 先天性结核 → P37.0：新生儿在母体内获得，不在本章
  • HIV造成的结核 → B20.0：这是ICD-10更新内容！当HIV与结核同时存在时，结核归类于B20.0而非A15-A19
  • 结核性硅沉着病 → J65：归入呼吸系统章节
  • 耐药结核 → 附加编码 U84.3：耐药肺结核可附加U84.3编码，说明对抗结核药物耐药

🫁 三、病毒性肝炎编码（B15-B19）——三维坐标下的编码迷宫

3.1 三大分类轴心

病毒性肝炎的编码是一个三维坐标系统：病程（急性/慢性）× 病原体（甲/乙/丙/丁/戊）× 临床表现（伴/不伴肝昏迷）。尤其是乙型肝炎，还多一个δ因子（丁肝）的维度。

病原急性编码慢性编码核心区分
甲肝 (HAV) B15.0(伴肝昏迷) / B15.9(不伴) —（不转为慢性） 甲肝不会慢性化，只有急性
乙肝 (HBV) B16.0(+δ+昏迷) / B16.1(+δ-昏迷) / B16.2(-δ+昏迷) / B16.9(-δ-昏迷) B18.0(伴δ) / B18.1(不伴δ) δ因子（丁肝）合并感染或重叠感染
丙肝 (HCV) B17.1 B18.2 丙肝无δ因子维度
丁肝 (HDV) 归类于乙肝伴δ因子的相关编码中 同左 丁肝=乙肝伴有δ因子，编码在乙肝中
戊肝 (HEV) B17.2 —（一般不自慢性化） 孕妇感染戊肝高危
未特指 B17.9 B18.9（慢性迁延性肝炎） B19（未特指病毒性肝炎）一般不做填报诊断

3.2 乙肝编码的δ因子迷宫——一张速查表搞定

1

判断病程：急性 or 慢性？

看 ALT、HBsAg、HBV-DNA、IgM抗HBc等指标。急性：病程<6个月，IgM抗HBc(+)；慢性：HBsAg持续>6个月。

2

判断δ因子（丁肝）

合并感染（co-infection）：HBV+HDV同时感染 → 急性乙肝+B16.0/B16.1；重叠感染（super-infection）：慢性HBV基础上感染HDV → B17.0（急性）或 B18.0（慢性）。

3

判断肝昏迷

伴肝昏迷 = 重型肝炎/肝衰竭。B15.0/B16.0/B16.2/B19.0 都表示伴有肝昏迷。

4

选择最终编码

按以上三维坐标，从速查表中选出唯一正确的编码。

3.3 ICD-10更新：乙肝的重大变更

⚠️ ICD-10更新要点：

  • Z22.5（病毒性肝炎病原携带者）已被删除。慢性乙肝携带者现编B18.1
  • 未特指急慢性的乙型肝炎，原默认B16.9（急性），更新后默认B18.1（慢性）
  • 妊娠期病毒性肝炎需区分妊娠期(O98.4)还是非妊娠期(B15-B19)

3.4 "既往乙肝"这个坑

临床医生常写"既往乙肝"或"乙肝病史"，编码员不能直接编码。必须查看化验结果：

化验结果可能编码含义
HBsAg(+) 且 HBV-DNA(+)、ALT持续升高B18.1慢性活动性乙型肝炎
HBsAg(+) 但 HBV-DNA(-)、ALT正常B18.1慢性乙肝携带者（原Z22.5已删除）
HBsAg(-) 且 抗HBs(+)Z86.1病毒性肝炎个人史（已痊愈）

🩸 四、HIV/AIDS 编码（B20-B24, Z21, R75）——四个临界状态

4.1 HIV编码的四个状态级别

HIV的编码不是"有就是B20"，而是根据确诊状态和并发症，精确区分四个级别：

编码状态判断依据特殊规则
B20-B24 确诊HIV病（有并发症） 经指定医疗机构确诊，且出现并发症 并发症不允许再分别编码（B20-B22 内部规则）
Z21 无症状HIV感染 确诊HIV但无任何临床症状 不能用作主诊断时填B20-B24
R75 HIV实验室证据 入院检查HIV抗体阳性，但未经指定医疗机构确诊 非确诊状态，仅筛查阳性
Z20.6 接触和暴露于HIV 有HIV暴露史但不确定感染 如职业暴露后观察

4.2 HIV并发症的三级嵌套规则（最容易绕晕的规则）

B20-B24 并发症编码规则（三句话记牢）：

① HIV病多个并发症存在于 B20-B22 同一类目下多个亚目时
   → 编码于该类目的 .7

② HIV病多个并发症存在于 B20-B22 中多个类目时
   → 编码于 B22.7

③ B20-B22中的其他码可作为附加诊断说明具体并发症

例：HIV + 肺结核 + 卡波西肉瘤

HIV病造成的结核 → B20.0

HIV病造成的卡波西肉瘤 → B21.0

并发症跨B20和B21两个类目 → 主诊断编码 B22.7

其他诊断附加 B20.0 和 B21.0 说明具体并发症

例：HIV + 卡波西肉瘤 + 淋巴瘤

两者都属于B21类目下 → 主诊断编码 B21.7

其他诊断附加 B21.0 和 B21.1

⚠️ HIV病发生前已存在的疾病：不允许分类在HIV病（B20-B24），应按原疾病所在章节编码。例如：患者患高血压10年，后感染HIV → 高血压仍编I10，不能套入B20-B24。

📋 五、肠道感染、败血症与病原体附加码

5.1 肠道传染病（A00-A09）——腹泻的分类迷宫

A00-A09覆盖了肠道感染的核心编码。但不同病因的腹泻归入不同类目——这需要编码员仔细阅读病历中关于病原体的信息：

病因类目"未指明具体病原"时的默认
细菌性A00-A05A04.9（未指明菌种）
原虫性A06-A07A07.9（未指明原虫）
病毒性A08A08.4（未指明病毒）
食物媒介（未指明感染）A05A05.9
其他传染性/未特指A09A09.0（感染性腹泻） / A09.9（新生儿腹泻）
非传染性K52.9不在本章！归消化系统

❌ 新生儿腹泻常见错误

所有新生儿腹泻都编 A09.9

新生儿非感染性腹泻应编 P78.3（围生期疾病章）；幼儿非感染性腹泻编 K52.9

✅ 正确区分逻辑

① 有无感染性病因？有→A00-A09

② 是否围生期？是→P78.3

③ 非感染性→K52.9

5.2 败血症、菌血症与脓毒症——三者的编码边界

这是临床上容易混用的三个概念，但ICD-10有明确区别：

概念编码临床定义主导词
菌血症 (Bacteremia) A49.9（未指明菌种） 病原体短暂侵入血流，迅速被防御系统清除，无或仅有轻微全身反应 菌血症
败血症 (Septicemia) A41.9（未指明菌种） 病原体侵入血液引起临床综合征，病情重于菌血症 败血症
脓毒症 (Sepsis) A41.9 感染致宿主反应失控→脏器功能障碍（SOFA≥2 或 qSOFA阳性） 毒血症→脓毒性→另见败血症
感染性休克 R57.2 脓毒症合并组织低灌注及器官功能障碍 休克→感染性
⚠️ 关键规则：脓毒血症与败血症的编码是同一编码（A41.9）。主导词路径：毒血症→脓毒性→另见败血症。明确菌种时编入更具体类目（如A41.0金黄色葡萄球菌败血症）。感染性休克（R57.2）通常作为附加编码，单独一个R码不能做主诊断。

5.3 B95-B98 病原体附加码——只能"陪跑"不能"主跑"

B95-B98是第1章最后的重要类目，用于标明分类于他处疾病的传染性病原体。这些码有一条铁律：

B95-B98 绝对不能作为主要编码，只能作为附加编码。

例：流感嗜血杆菌引起的急性鼻窦炎

主诊断：J01.9（急性鼻窦炎）

附加编码：B96.3（流感嗜血杆菌）

J01.9 在前，B96.3 在后

例：幽门螺旋杆菌感染

注意：原 B96.8 已更新为 B98.0

字典没有B98，属于ICD-10更新内容

5.4 寄生虫病（B65-B83）

寄生虫病以"侵染"为主导词。编码轴心为虫的类型，部分编码细致到界门纲目科属种。未指明寄生虫病编B89。

寄生虫类型类目代表疾病
原虫病（Protozoal）B50-B64疟疾(B50-B54)、利什曼病(B55)
蠕虫病（Helminthiases）B65-B83血吸虫病(B65)、绦虫病(B68)、蛔虫病(B77)
虱病/螨病/其他B85-B89疥疮(B86)

💡 六、重难点深度解析：A15还是A16？——看不见的"证实线"

场景还原：一个最常见的编码失误

临床医生出具"出院诊断：继发性肺结核"，病程记录中记载了"痰涂片抗酸染色阳性"和"GeneXpert MTB/RIF检出结核分枝杆菌"。编码员查索引得到A16.2，直接编码上报。这个编码错了。

错在哪里？

因为A15和A16的分界线是一道编码员必须穿透"诊断表面"去看化验单的线——只看临床诊断文字不足以判断。

1

看诊断名称只能知道"是结核"

继发性肺结核、浸润性肺结核、空洞型肺结核……这些诊断名告诉你病变部位和类型，但不告诉你"证实状态"。

2

看化验报告才能知道A15还是A16

痰涂片(+)、痰培养(+)、病理活检(+)、GeneXpert(+) → A15

上述均(-)或未做→ A16

3

看索引才知道：默认走向是A16

索引中"结核→肺"在不指定"经证实"修饰词时，给出的就是A16。所以不看化验报告直接用索引编码=永远编A16=大概率错！

本案例的正确编码流程

临床诊断：继发性肺结核
化验报告：痰涂片抗酸染色(+) ，GeneXpert MTB/RIF(+)

步骤1：确定主导词 → 结核
步骤2：查索引 → 结核 → 肺 → "经显微镜下痰检查证实" → A15.0
步骤3：核对卷一 → A15.0 "肺结核，经痰液镜检证实"
            → 包括"伴有或不伴有痰培养"
            → GeneXpert属分子生物学证实，也归入已证实类
            → 不包括：HIV造成的结核(B20.0) —— 本例患者HIV(-)，确认无此交叉

最终编码：A15.000 继发性肺结核（国临版扩展码）
⚠️ 这个错误有多严重？A15和A16在DRG分组中可能落入不同ADRG，直接影响医院收入。对于结核专科医院，A15 vs A16的编码准确率是质控检查的硬指标。

1
基础题 · 结核证实状态判断 PRACTICE

患者出院诊断为"空洞型肺结核"，病程记录：痰涂片抗酸染色(-)，痰培养：结核分枝杆菌生长(+)。

问题：应编码A15还是A16？具体亚目是什么？

✅ 编码：A15.1（肺结核，仅经痰培养证实）

推理过程：

① 痰涂片(-)排除了A15.0

② 痰培养(+) → 属于细菌学证实 → 归入A15（不是A16）

③ 仅经痰培养证实（涂片阴性+培养阳性）→ 精确匹配A15.1

💡 核心要点：不是"涂片(+)才叫A15"，只要任何一种细菌学/组织学方法证实阳性，就是A15。涂片(-)培养(+)仍然编码A15.1而非A16。

2
基础题 · 病毒性肝炎三维编码 PRACTICE

35岁男性，因"乏力、纳差、尿黄1周"入院。检验结果：HBsAg(+)，抗HBc IgM(+)，HBeAg(+)，HBV-DNA 3.2×10⁷ IU/mL，HDV抗体(-)，ALT 1250 U/L，TBil 68 μmol/L。神志清楚，无肝性脑病表现。

问题：请给出完整的编码和推理过程。

点击查看答案

✅ 编码：B16.9（急性乙型肝炎，不伴有δ因子，不伴有肝昏迷）

推理过程：

① 病程判断：抗HBc IgM(+) → 急性感染（<6个月）→ 急性乙型肝炎，编码路径 B16

② δ因子判断：HDV抗体(-) → 不伴有δ因子 → 排除B16.0/B16.1

③ 肝昏迷判断：神志清楚，无肝性脑病 → 不伴有肝昏迷 → 排除B16.2 → 编码B16.9

💡 口诀：乙肝四步走——急慢性 → δ因子 → 肝昏迷 → 定编码

3
基础题 · HIV状态区分 PRACTICE

判断以下三个场景各应编什么码：

场景描述你的编码
A患者因肺炎入院，既往在外院确诊HIV，目前CD4 580 cells/μL，无任何HIV相关症状？
B体检发现HIV抗体阳性，转送至疾控中心确诊中，目前仅有初筛结果？
C因卡氏肺孢子虫肺炎入院，已确诊HIV，CD4 85 cells/μL？
点击查看答案

✅ A：应编码 Z21（无症状HIV感染状态）

理由：已确诊HIV但无临床症状（CD4正常、无症状）→ Z21。肺炎与HIV无关，另外编码肺炎。

✅ B：应编码 R75（HIV实验室证据）

理由：仅有实验室初筛阳性，未经指定医疗机构确诊→ R75，而非Z21或B20-B24。

✅ C：应编码 B20.6（HIV病造成的卡氏肺孢子虫肺炎）

理由：确诊HIV + 机会性感染（卡氏肺孢子虫肺炎）→ 肺炎纳入B20-B24，不允许再单独编码J17.3*。

进阶题 · 败血症+感染性休克综合判断 CHALLENGE

76岁女性，因高热、意识模糊入院。血培养回报：大肠埃希菌(E.coli)。患者出现SBP 85 mmHg、呼吸频率26次/min、SOFA评分增加3分。临床诊断：①大肠埃希菌败血症 ②感染性休克。

问题：主诊断应编什么？需要几个编码？注射用抗生素已确认耐药？

点击查看答案

✅ 主诊断：A41.5（大肠埃希菌败血症）

✅ 其他诊断：R57.2（感染性休克）——作为附加编码

推理过程：

① 血培养阳性 + 明确菌种(E.coli) → A41.5（非A41.9未特指）

② 感染性休克是败血症的严重并发症 → R57.2作为附加编码

③ 如果患者对多种抗生素耐药 → 可附加编码耐药相关码

💡 编码顺序：A41.5（主诊断）→ R57.2（附加诊断，说明休克）

进阶题 · HIV多并发症嵌套规则 CHALLENGE

确诊HIV患者，同时患有以下并发症：①卡波西肉瘤（皮肤） ②HIV脑病 ③口腔念珠菌病。既往有原发性高血压病史5年。

问题：请给出完整的编码方案（主诊断+其他诊断）。

点击查看答案

✅ 完整编码方案：

主诊断：B22.7（HIV病造成的多处分类于他处的疾病）

其他诊断：

① B21.0（HIV病造成的卡波西肉瘤）

② B22.0（HIV病造成的脑病）

③ B20.4（HIV病造成的念珠菌病）

④ I10（原发性高血压）——HIV前已存在，不纳入B20-B24

推理过程：

① 三个并发症分别属于B21、B22、B20三个不同类目 → 按嵌套规则编码B22.7

② 其他诊断附加B21.0、B22.0、B20.4说明具体并发症

③ 高血压为HIV前已存在的疾病 → 不使用B20-B24，仍编I10

进阶题 · 交叉章节鉴别——感染放哪里？ CHALLENGE

判断以下诊断应归入第1章（A00-B99）还是其他章节：

诊断你的判断编码
① 细菌性肺炎（肺炎链球菌）第1章 / 其他章？
② 妊娠28周合并泌尿系感染（大肠埃希菌）第1章 / 其他章？
③ 新生儿（日龄3天）败血症（大肠埃希菌）第1章 / 其他章？
④ 急性肾盂肾炎第1章 / 其他章？
点击查看答案

✅ ① 其他章 → J13（肺炎链球菌性肺炎）

理由：呼吸道感染的细菌性肺炎归入呼吸系统J00-J22，不在第1章

✅ ② 其他章 → O23.-（妊娠期泌尿生殖道感染）

理由：妊娠期感染 → 第15章O编码优先

✅ ③ 其他章 → P36.4（新生儿大肠埃希菌败血症）

理由：围生期感染（出生后7天内）→ 第16章P编码优先

✅ ④ 其他章 → N10（急性肾盂肾炎）

理由：局部器官感染（非全身播散性）→ 归入泌尿系统章节

💡 核心口诀：局部感染看器官，妊娠围生期优先，只有全身性/播散性/可传播的感染才归第1章。

✅ 八、今日重点回顾

第1章是特殊组合章：以病因为轴心，非解剖部位。需要判断"该不该放进这一章"
A15 vs A16的分界线是"细菌学/组织学证实"——必须看化验报告
病毒性肝炎三维编码：病程(急/慢) × 病原体 × 肝昏迷，乙肝多加δ因子维度
HIV四个状态级别：B20-B24(确诊+并发症) → Z21(无症状) → R75(未确诊) → Z20.6(暴露)
未明确慢性→默认急性：传染病默认活动性/急性（本章特有规则）
B95-B98绝对不做主诊断：病原体附加码只能陪跑
五大"不包括"：Z22携带者、O98妊娠感染、P35-P39围生期、J00-J22呼吸道、局部感染
败血症A41.9 = 脓毒症A41.9，感染性休克R57.2作为附加编码
