📋 一、病案首页到底是什么？

1.1 官方定义

根据《住院病案首页数据填写质量规范（暂行）》第二条：

住院病案首页是医务人员使用文字、符号、代码、数字等方式，将患者住院期间相关信息精炼汇总在特定的表格中，形成的病例数据摘要。包括：① 患者基本信息、② 住院过程信息、③ 诊疗信息、④ 费用信息。

1.2 为什么编码员必须吃透首页？

三个字：入口、出口、证据。

角色首页的作用编码员关联
入口 首页是 DRG/DIP 分组器读取的唯一数据源 你编的每一个 ICD 码，都直接影响入组
出口 首页是医保结算明细的依据，决定医院收入 离院方式填错=结算类型错误=扣款退费
证据 首页是医疗纠纷、飞检、国考中的核心证据材料 你的编码被追溯审计，责任终身制
⚠️ 一条数据的蝴蝶效应：

主诊断编码错一位 → DRG 入错组 → 权重降 0.3 → 单病例结算亏损 3000 元 → 全院年亏损可能达数百万。这就是为什么编码员的工作叫"小数点后定乾坤"。

🏗️ 二、病案首页四大板块深度拆解

第一部分
患者基本信息
姓名、性别、出生日期、年龄、国籍、民族、身份证号、婚姻状况、职业、出生地、籍贯、工作单位、地址、电话、邮编、联系人信息……
→ 主要由入院登记处/住院处填写，编码员一般不涉及
第二部分
住院过程信息
入院日期、入院科别、转科情况、出院日期、出院科别、实际住院天数、入院途径（门诊/急诊/转诊）、入院前经急诊留观天数、离院方式、是否有出院31天内再住院计划、颅脑损伤患者昏迷时间……
→ 编码员重点关注：离院方式（直接影响结算）、实际住院天数（影响 DRG 时间效率指标）
第三部分★核心
诊疗信息（编码员的"主战场"）
主要诊断（1个，ICD-10编码 + 名称）、 其他诊断（可多个，ICD-10编码 + 名称 + 入院病情代码）、 主要手术/操作（ICD-9-CM-3编码 + 名称 + 日期 + 术者）、 其他手术/操作（可多个）、 病理诊断、损伤中毒外因、药物过敏、血型……
→ 编码员100%负责的内容，是编码错误的高发区
第四部分
费用信息
总费用 + 21项分类费用（床位、护理、检查、化验、药品、耗材、手术、麻醉等）
→ 主要由财务/HIS自动生成，但编码员需确保费用分类与手术/操作编码逻辑一致（如：有手术编码则手术费不应为0）
编码员记忆口诀："一基本不碰，二只看离院，三是主战场，四要扫一眼。"

📝 三、诊疗信息——编码员的主战场的 5 条铁律

3.1 主要诊断：只有一个，但最难选

规则说明反面案例
① 三最原则 对患者健康危害最大、消耗医疗资源最多、住院时间最长 把"高血压3级"当主诊断而忽略了"急性心梗"
② 病因优先症状 明确病因后，用病因码替代症状码 MRI 确诊脑梗死后仍用 R42（头晕）做主诊断
③ 手术导向 有手术/操作时，主诊断必须与主手术匹配 手术做了胆囊切除术，主诊断却填了高血压
④ 治疗目的 如为专项治疗入院（放化疗/康复/透析），以治疗目的为主诊断 来化疗的却把肿瘤本身写在主诊断位置
⑤ 合并编码优先 ICD 提供了合并编码时，使用合并编码充当主诊断 高血压性心脏病分别编了 I10 和 I51.8

3.2 其他诊断：不是"多余的"，是"不能少的"

其他诊断 = 除主诊断外的所有诊断，按严重程度降序排列。每个其他诊断还必须填写入院病情代码：

代码含义举例常见错误
1有明确诊断，入院时已有入院即有高血压3级把入院新发现的也填成1
2入院时未明确，住院期间通过检查新发现的住院CT发现肺结节把入院已知的填成2
3住院期间新发生的（并发症/院内感染）术后发生切口感染漏填术后并发症
4入院时已有，但本次住院期间未进一步明确或未作特殊处理陈旧性脑梗死后遗症，本次未处理填成1或遗漏
⚠️ 入院病情代码是 DRG 并发症/合并症（CC/MCC）判定的关键依据：代码 3（住院期间新发生）的疾病可能触发更高的 DRG 权重——但也可能被飞检盯上，如果实际上不是住院期间发生的却填了 3，就是"高编诊断"。

3.3 手术/操作：谁主谁次？

手术操作按手术级别（1-4级）降序排列，同一级别的按手术日期先后排列。主要手术 = 级别最高、风险最大、消耗资源最多的那一个。

编码员注意：诊断性操作（如胃镜、气管镜）如果有治疗性手术，应排在治疗性手术之后。但如果是唯一的操作，则填写在此栏。

🚪 四、离院方式——6 种代码一张表搞定

离院方式是 2011 年首页修订后新增的必填指标，直接影响医保结算类型、30天再住院率统计、分级诊疗考核。错误率据研究高达 15%以上。

代码名称核心判定条件典型场景
1医嘱离院病情稳定达到出院标准，遵医嘱出院急性阑尾炎术后恢复良好出院
2医嘱转院本院诊疗条件不足，医嘱转往其他医疗机构继续治疗心梗患者本院无介入条件，转往上级医院
3医嘱转社区卫生服务机构/乡镇卫生院病情稳定后转往基层医疗机构康复脑梗恢复期患者转社区康复
4非医嘱离院患者/家属自行要求离院，未达到出院标准AAD（自动出院），家属签字放弃治疗
5死亡住院期间患者死亡经抢救无效宣布临床死亡
9其他以上5种以外的特殊情况（极少使用）被司法机关带走等极特殊情况

4.1 最容易踩的三个边界辨析

❌ 误区1：转科 = 转院？

患者从神经内科转入康复科 → 离院方式不能填"2（转院）"！转科是院内转移，离院方式要看最终出院时的去向。内科→康复科→医嘱离院 = 代码1。

✅ 正确

转院是指离开本院去其他独立医疗机构。院内转科不改变离院方式。

❌ 误区2：死亡 = 所有未治愈出院的都填死亡？

患者病重，家属签字放弃治疗自动出院，3天后在家中死亡 → 离院方式应该填"4（非医嘱离院）"，不是"5（死亡）"。死亡必须发生在住院期间。

✅ 正确

离院方式记录的是出院那一刻的状态，不是出院后发生了什么。患者活着离开医院，就是非医嘱离院。

❌ 误区3：医嘱离院 = 治愈？

很多编码员认为只有"治愈"才能填"医嘱离院"。实际上，病情稳定、达到出院标准即可——有好转/未愈也可以遵医嘱出院。

✅ 正确

离院方式与出院情况（治愈/好转/未愈/死亡/其他）是两个独立字段。医嘱离院不看是否治愈，只看是否遵医嘱。

🔗 五、首页与 DRG/DIP 的数据流向

理解下面这条链路，就知道为什么首页上的每一个编码都值钱：

临床医生写诊断
→
编码员翻译成 ICD 码
→
录入病案首页
上传医保平台
→
分组器读取
→
判定 MDC → ADRG → DRG
计算权重 × 费率 = 结算金额

其中编码员能直接影响的是前三个环节：

  • 主诊断 ICD 码 → 决定进入哪个 MDC（26个大类之一）
  • 主手术 ICD-9-CM-3 码 → 在 MDC 内决定 ADRG
  • 其他诊断（特别是入院病情=3的并发症）→ 可能触发 CC/MCC，提升 DRG 权重
  • 离院方式 → 死亡（5）和非医嘱离院（4）的病例 DRG 权重可能不同
⚠️ QY 病例（歧义病例）警告：如果主诊断和主手术不匹配——比如主诊断在呼吸系统但主手术是消化系统的——分组器无法确定入哪个 ADRG，病例被标记为 QY，结算打折。这是编码员必须避免的。

  5. 主要手术操作编码

7 → D（费用信息）

2
基础题 · 入院病情代码判定 PRACTICE

为以下场景选择正确的入院病情代码（1-4）：

场景诊断入院时已存在？住院期间怎么发现的？代码
A高血压3级是，既往有明确病史入院时即有记录？
B肺结节入院时不知道住院期间胸部CT意外发现？
C切口感染否，手术后出现术后第3天切口有渗液？
D陈旧性腔梗是，5年前的病史本次住院未做特殊处理？
点击查看答案

✅ A → 1（有明确诊断，入院时已有）

✅ B → 2（入院时未明确，住院检查新发现的）。注意：B 如果发现后做了进一步处理（如穿刺活检），仍填2，因为这代表"本次新发现"的状态。

✅ C → 3（住院期间新发生的并发症）——这是最敏感的代码，触发 DRG 并发症/合并症。

✅ D → 4（入院时已有，但本次未进一步明确或处理）。注意：D 表明这个诊断"存在但本次不关心"，不影响本次资源消耗。

3
基础题 · 离院方式场景判断 PRACTICE

判断以下各病例的离院方式代码：

  1. 急性阑尾炎术后第3天，恢复良好，医生查房后开具出院医嘱
  2. 大面积心梗，本院无导管室介入条件，医生联系后转往省人民医院
  3. 脑梗恢复期，转往社区卫生服务中心继续康复治疗
  4. 晚期肿瘤患者，家属签字放弃进一步治疗，自动出院（出院时仍有生命体征）
  5. 多发伤，住院第5天经抢救无效死亡
点击查看答案

✅ 1 → 代码 1（医嘱离院）：医生主动开具出院医嘱，患者达到出院标准。

✅ 2 → 代码 2（医嘱转院）：关键——是"医嘱"转院（不是患者自己要求），且转往其他独立医疗机构。

✅ 3 → 代码 3（医嘱转社区）：注意与"2"的区别——转院是去其他医院，转社区是去基层。

✅ 4 → 代码 4（非医嘱离院）：不是代码 5！患者出院时仍有生命体征，死亡发生在出院后。

✅ 5 → 代码 5（死亡）：必须在住院期间发生死亡。

4
进阶题 · 主诊断+离院方式综合 CHALLENGE

患者因"腹痛待查"入院，住院期间胃镜+病理确诊"胃窦部低分化腺癌"。出院时患者家属要求转往肿瘤医院进一步治疗。

问题：

  1. 主诊断应该用什么？
  2. 离院方式代码是什么？
  3. 入院病情代码（针对肿瘤诊断）是什么？
点击查看答案

✅ 1. 主诊断：C16.3 胃窦恶性肿瘤。因为有明确病理确诊，不能用症状码 R10.4（腹痛）。

✅ 2. 离院方式：代码 2（医嘱转院）。注意——家属要求转院，医生同意并开了转院医嘱 → 是"医嘱转院"而非"非医嘱离院"。

✅ 3. 入院病情代码：2——入院时未明确诊断（只知道腹痛），住院期间通过胃镜+病理新发现的。注意不能填 1（入院时并未确诊腺癌）。

5
进阶题 · 首页+DRG联动 CHALLENGE

某病例：主诊断 J44.1（慢性阻塞性肺病伴急性加重），其他诊断含 N39.0（泌尿系感染，入院病情=3），主手术 96.59（其他支气管灌洗）。

问题：

  1. 这个病例会进入哪个 MDC？（A.呼吸系统 B.泌尿系统 C.循环系统）
  2. 泌尿系感染（入院病情=3）会对 DRG 分组产生什么影响？
  3. 如果主诊断被错误填成 N39.0 而 J44.1 被放在其他诊断，会发生什么？
点击查看答案

✅ 1. → A（呼吸系统 MDC），因为 MDC 由主诊断决定，J44.1 属于第 X 章呼吸系统疾病。

✅ 2. 泌尿系感染（入院病情=3）作为住院期间新发生的并发症，可能在 DRG 分组中触发CC（并发症），将病例从无并发症组提升到有并发症组，DRG 权重因此增加。

✅ 3. 如果主诊断被错误填成 N39.0：病例会进入泌尿系统 MDC（而非呼吸系统），而主手术 96.59（支气管灌洗）属于呼吸系统操作——导致诊断与手术不在同一 MDC，分组器无法匹配 → QY 病例（歧义），结算大幅打折。

6
进阶题 · 边界情景综合判断 CHALLENGE

某车祸多发伤患者，入院时诊断：① 脾破裂；② 左侧多发肋骨骨折；③ 左肺挫伤。

住院期间行急诊脾切除术，术后在ICU监护3天。术后第5天患者出现肺部感染（痰培养铜绿假单胞菌阳性）。第12天患者病情稳定，医生建议转往康复医院，但患者家属坚决要求回家休养，签字自动出院。

请完整回答：

  1. 主诊断是什么？主要手术是什么？
  2. 肺部感染是否应该填入其他诊断？入院病情代码填什么？
  3. 肋骨骨折和肺挫伤需要填入其他诊断吗？入院病情代码分别是什么？
  4. 离院方式填什么代码？
点击查看答案

✅ 1. 主诊断：S36.0 脾损伤/破裂（损伤章，手术导向→脾切除与脾破裂完全匹配）。主手术：41.5 脾切除术。

✅ 2. 肺部感染应填入其他诊断，编码 J15.1（铜绿假单胞菌性肺炎）。入院病情代码 = 3——因为入院时没有肺部感染，是住院第5天新发生的（院内感染）。这个代码3将触发 DRG 的并发症标记。

✅ 3. 肋骨骨折和肺挫伤应填入其他诊断。编码 S22.3（肋骨骨折）、S27.3（肺挫伤）。入院病情代码均 = 1——因为这些损伤入院时已存在。

✅ 4. 离院方式 = 代码 4（非医嘱离院）。医生建议转院但家属坚决要求回家、签字自动出院 → 未达到出院标准但自行离院。

排名错误类型具体表现正确做法
1 主诊断与主手术不匹配 主诊断是呼吸系统疾病、主手术是消化系统手术 → QY 病例 有手术时主诊断必须跟着手术走；先定手术，再定诊断
2 离院方式填错 转院填成医嘱离院（1）；自动出院填成死亡（5）；转社区填成转院（2） 背熟6个代码的判定核心："是否医嘱 → 去向到哪里"
3 入院病情代码全填"1" 不管三七二十一，所有其他诊断的入院病情都填1 逐一核对：这个诊断入院时到底存不存在？
4 漏填并发症 住院期间发生的切口感染、院内感染、药物不良反应等不填 入院病情=3 的并发症是"金矿"——合理填入可提升 DRG 权重
5 诊断排序错误 把轻微的慢性病排在急性重病前面 按严重程度降序排列，主诊断之后跟最重的其他诊断

✅ 八、今日重点回顾

首页分四大块：基本→过程→诊疗→费用，编码员主战场在第三块
主诊断三最原则：危害最大、资源最多、时间最长
入院病情代码 1-4 各有严格界定，尤其 3（住院期间新发生）最敏感
离院方式 6 个代码：按"是否医嘱→去向"两级判断
离院时状态决定代码，非医嘱离院≠死亡，转科≠转院
主诊断→MDC，主手术→ADRG，其他诊断→CC/MCC→DRG权重
QY 病例 = 主诊断和主手术跨 MDC 不匹配，结果=结算打折
术后并发症（入院病情=3）合理填入可合法提升DRG权重
