🗺️ 一、H章全景速览：两大分部 + 14个板块

👁️ 二、白内障编码 H25-H28 —— 三维决策体系深度拆解

白内障是 H 章最核心的编码板块之一，也是最容易出错的。编码员不能看到"白内障"就编 H25.9（未特指），因为 ICD-10 已经为白内障建立了"病因+部位+形态"三维编码体系。

2.1 为什么白内障编码是"三维立体"的？

白内障不是一个病，而是一个临床表现——它背后的病因决定了归类到哪个类目。ICD-10 的编码体系按病因将白内障分为四个核心类目：

类目病因分类是什么典型场景
H25老年性白内障最常见的退行性晶状体混浊，年龄＞50岁且无明确其他病因"双眼老年性白内障""老年性核性白内障"
H26.0幼年性白内障儿童/青少年发生，无明确代谢/外伤等继发因素"先天性白内障"如无遗传依据可编此码（注意与Q12.0区分）
H26.1外伤性白内障明确由外伤导致（穿孔伤、挫伤、放射伤等）"外伤性白内障"附加外因 S05/V 码
H26.2并发性白内障由其他眼部疾病（如葡萄膜炎、高度近视、视网膜脱离）继发先查原发病编码，再编 H26.2
H26.3药物性白内障由糖皮质激素等药物长期使用引起需附加外因编码（Y42-Y59）
H28*全身疾病所致白内障糖尿病性白内障（H28.0*）、甲状旁腺功能减退性（H28.1*）等星号码，必须与病因剑号码联动
Q12.0先天性白内障出生即存在或在生命早期出现不属于H章，归入Q章（先天性畸形）

2.2 白内障编码四步决策法

1

判断发病年龄和先天/后天

出生即有 → Q12.0（Q章，不在H章内）；后天获得 → 继续第2步

易错点：临床写"先天性白内障"但实际没有遗传/发育证据的幼年性白内障，应归 H26.0，不是 Q12.0

2

判断病因：是眼部疾病还是全身疾病？

全身疾病（糖尿病、甲状旁腺功能减退等）→ H28.0* 或 H28.1* 或 H28.2*（星号，必须配剑号）

眼部本身疾病继发（葡萄膜炎、视网膜脱离）→ H26.2（并发性白内障）

核心理念：H28* 是"身体其他部位病了→眼睛受牵连"，H26.2 是"眼睛自己病了→晶状体受牵连"

3

判断是否为外伤/药物所致

明确外伤史 → H26.1（外伤性白内障），附加 S05 损伤外因 V/Y 编码

明确药物史（如长期糖皮质激素）→ H26.3（药物性白内障），附加 Y42-Y59 药物外因

4

排除上述原因后，按年龄和形态编细目

50岁以上、退行性改变 → H25（老年性），再按形态细分亚目

儿童/青少年、无特殊病因 → H26.0（幼年性）

2.3 老年性白内障 H25 的形态亚目拆解

亚目形态编码特征
H25.0老年性初期白内障晶状体皮质轻度混浊，尚未明显影响视力
H25.1老年性核性白内障混浊集中于晶状体核（中央区），最常见类型
H25.2老年性白内障，莫加尼（Morgagni）型过熟期白内障，晶状体核下沉
H25.8其他老年性白内障混合型、囊膜型等不典型形态
H25.9未特指的老年性白内障尽量避免使用，只有在病程记录中完全没有形态描述时才用
⚠️ 编码陷阱——H25.9 滥用：很多编码员看到"老年性白内障"就直接编 H25.9。实际上，只要病程记录或裂隙灯检查报告提到了"核性""皮质性""后囊下"等形态描述，就应该编到对应的细目。H25.9 是信息不足时的兜底选项，不是默认值。

2.4 糖尿病性白内障的星剑号编码

糖尿病性白内障是考试高频考点，编码时必须星剑号联动：

临床诊断：2型糖尿病性白内障
主导词：白内障 → 见于 → 糖尿病 → 索引指向 H28.0*

编码结果：
主诊断：E11.3†  2型糖尿病伴有眼部并发症（剑号——病因）
附加码：H28.0*  糖尿病性白内障（星号——眼部表现）

❌ 错误：只编 H28.0* 而不编 E11.3† → 丢失病因编码，DRG入组偏差
❌ 错误：只编 H25.x + E11.9 → 没有用星剑号，属于编码规则执行不到位

📐 三、青光眼编码 H40-H42 —— "轴心在于房角"

青光眼的编码体系以"房角状态"为核心分类轴心，同时结合"原发性 vs 继发性"和"急性 vs 慢性"。这是与白内障完全不同的分类逻辑。

3.1 为什么青光眼编码不能只看诊断名称？

临床诊断常写"青光眼""原发性青光眼""继发性青光眼"，但这些名称在 ICD-10 中只是分类的一个维度。完整的青光眼编码需要三个维度的信息：

❌ 仅凭诊断名称

看到"青光眼" → 编 H40.9（未特指）
看到"原发性青光眼" → 编 H40.0（可疑）
→ 大概率编错！

✅ 需要三重信息

① 房角状态：开角型 / 闭角型（必须看房角镜报告）
② 原发/继发：原发性 / 继发于其他疾病
③ 急/慢性：急性发作 / 慢性进展

3.2 H40 亚目全表——决策树

亚目分类临床特征关键鉴别点
H40.0可疑青光眼眼压偏高但无视野缺损和视盘损害不是确诊的青光眼！如果已确诊不应编此码
H40.1原发性开角型青光眼房角开放但房水排出不畅，慢性进展房角镜报告："房角开放，宽角" → 排除闭角型
H40.2原发性闭角型青光眼房角狭窄/关闭，房水排出受阻房角镜报告："房角狭窄，N1-N2" → 闭角型
H40.3继发于眼外伤的青光眼外伤后房角后退、前房积血导致眼压升高需附加外伤外因 S05 + V/Y 码
H40.4继发于眼部炎症的青光眼葡萄膜炎、角膜炎等炎症继发需同时编原发炎症码
H40.5继发于其他眼部疾病的青光眼晶体膨胀、新生血管等新生血管性青光眼即归此类
H40.6药物性青光眼糖皮质激素等长期使用导致附加药物外因 Y42-Y59
H42.0*内分泌/营养/代谢疾病青光眼糖尿病、甲状腺疾病等全身病在眼部的表现星号码，必须配合剑号（如 E11.3†）
⚠️ H40.0（可疑青光眼）最容易用错：这是筛选/疑似状态，仅用于"眼压高但未确诊"的观察阶段。一旦通过视野+OCT+房角镜明确了青光眼诊断，必须从 H40.1 起步。编码员的职责是追问"到底是不是确诊的"，而不是看到"青光眼"就直接给码。

3.3 开角型 vs 闭角型的核对流程

以"原发性青光眼，房角镜检查提示房角开放"为例：

主导词：青光眼 → 原发性
  └ 开角型 → 索引指向 H40.1

核对卷一 H40.1：
  包括：原发性开角型青光眼（慢性单纯性青光眼）
  不包括：可疑青光眼（H40.0）、绝对期青光眼（H44.5）

✅ 最终编码：H40.1 原发性开角型青光眼

如果没有房角镜报告 → 编码员应联系医生确认房角状态
绝不能自己猜测"大概率是开角/闭角"然后随意编码

👂 四、中耳炎编码 H65-H75 —— "四象限"分类法

中耳炎编码是 H 章中的第二大核心板块，因为中耳炎是耳鼻喉科最常见的诊断之一。ICD-10 对中耳炎的分类可以用一个"四象限"矩阵来概括：

中耳炎编码四象限决策矩阵

🔴 急性 + 化脓性
H66.0 急性化脓性中耳炎
典型表现：耳痛、发热、鼓膜充血穿孔、脓性分泌
🔵 急性 + 非化脓性
H65.0 急性浆液性中耳炎
典型表现：耳闷、传导性聋、鼓膜内陷、积液无脓
🟣 慢性 + 化脓性
H66.1 慢性咽鼓管鼓室化脓性中耳炎
H66.2 慢性上鼓室-鼓窦化脓性
典型表现：反复流脓、鼓膜穿孔、听力下降
🟢 慢性 + 非化脓性
H65.2 慢性浆液性
H65.3 慢性黏液样
H65.4 其他慢性非化脓性
典型表现：分泌性中耳炎、胶耳

4.1 化脓性中耳炎 H66 的亚目细节

编码类型关键解剖部位临床关联
H66.0急性化脓性中耳（鼓室）耳痛+脓性分泌物+发热，首次发作或急性加重
H66.1慢性咽鼓管鼓室化脓性鼓室（咽鼓管-鼓室通道）鼓膜中央穿孔，反复流脓，良性型
H66.2慢性上鼓室-鼓窦化脓性上鼓室+鼓窦边缘性穿孔，伴胆脂瘤风险，危险型
H66.3其他慢性化脓性其他部位或混合不能明确归入上述两类的慢性化脓性中耳炎
H66.4未特指的化脓性中耳炎—仅在无分型信息时使用
H66.9未特指的中耳炎—诊断过于笼统，尽量避免
💡 慢性化脓性中耳炎分型的关键：H66.1（良性型，咽鼓管鼓室）和 H66.2（危险型，上鼓室鼓窦）的区分依赖于耳内镜报告中对穿孔位置的描述——中央穿孔→H66.1，边缘/上鼓室穿孔→H66.2。这是临床报告和编码衔接的典型案例。

4.2 胆脂瘤 H71 —— 不是中耳炎但常相伴出现

编码描述编码要点
H71中耳胆脂瘤非肿瘤性角化鳞状上皮在中耳的异常生长，常与慢性中耳炎共存
H95.0乳突切除术后复发胆脂瘤术后状态的特殊编码，与 H71 互斥

5.2 糖尿病眼病编码——最难决策的星剑号场景

糖尿病眼病编码的决策路径：

1

是否有视网膜病变？

有 → E1x.3† + H36.0*（糖尿病伴有眼的并发症，剑号+星号）

只有白内障、无视网膜病变 → 继续第2步

2

是否同时有视网膜病变+白内障？

有 → 关键决策：E11.3（伴有眼部并发症）还是 E11.7（伴有多个并发症）？

只有眼部并发症 → E11.3† + H36.0* + H28.0*

眼部+其他系统（如肾、神经）→ E11.7，但注意 E11.7 的局限性

3

E11.3 vs E11.7 的抉择

E11.3†：仅眼部并发症，保留剑号，可带动多个星号（H36.0*+H28.0*）

E11.7：多个系统并发症（眼+肾/神经等），但不带剑号，星号码需要单独处理

⚠️ E11.7 的陷阱：E11.7（2型糖尿病伴有多个并发症）不自带剑号，这意味着使用 E11.7 时失去了带动星号码的"挂钩"。如果患者有多个系统并发症且其中有眼病，正确的做法可能是：E11.7（主诊断，多并发症）+ H36.0*（视网膜病变）+ H28.0*（白内障），三个码全部需要。但要确认国临版的星剑号规则是否强制要求剑号联动。

💡 重难点深度解析：青光眼开角型 vs 闭角型——"不能靠猜"的编码红线

为什么这个区分编码员反复犯错？

青光眼的"开角型/闭角型"是 H 章中最高频的编码错误源之一。错误的根源不在编码规则本身，而在于编码员试图从诊断文字中推断，而不是从检查报告中确认。

真实场景对比：为何容易掉坑

❌ 依赖诊断名称推断

场景：诊断写"原发性青光眼"，没有注明开角/闭角

编码员想："青光眼大部分是开角型，就编 H40.1 吧"

结果：实际房角镜检查是闭角型（N2），该患者应编 H40.2。编码错误导致统计数据中"开角/闭角"比例失真。

✅ 根据检查报告确认

场景：诊断写"原发性青光眼"，房角镜报告："双眼房角狭窄，N2~N3"

编码员看到"N2~N3"→ 房角狭窄 → 闭角型 → H40.2

结果：编码与临床检查吻合，数据准确，DRG 入组无误。

编码员如何处理"诊断信息不全"的情况？

当诊断仅写"青光眼"但缺少分型信息时：

1️⃣ 查看房角镜检查报告
   → 关键字："房角开放/宽角" → H40.1
   → 关键字："房角狭窄/关闭" → H40.2
   → 关键字："房角后退" → H40.3（外伤继发）

2️⃣ 查看 OCT 报告
   → "RNFL变薄 + 视野缺损" → 已确诊（排除 H40.0）

3️⃣ 如果以上都没有 → 主动联系主管医生确认
   → 不要自编 H40.9（未特指）
   → 因为"开角/闭角"是治疗方案选择的基础（开角用药物/激光，闭角首选手术）
   → 编码员有责任确保诊断分型信息完整
⚠️ 编码红线："未特指"码（H40.9）在青光眼编码中是高频滥用码。ICD-10 设计"未特指"的本意是给"确实无法确定"的极少数情况预留出口，而不是给编码员偷懒的快捷通道。每当你想用 H40.9 时，扪心自问：我真的已经穷尽了所有获取信息的途径了吗？

急性闭角型青光眼发作——是否需要单独编码"急性"？

临床诊断常写"急性闭角型青光眼"，但在 ICD-10 中，H40.2（原发性闭角型青光眼）已经包含了急性、亚急性、慢性所有阶段。急性发作不改变编码——因为亚目已经按房角状态（闭角型）而不是按病程（急性/慢性）来分类了。

基础题 · 青光眼开角/闭角鉴别 PRACTICE

某眼科门诊患者，诊断写"原发性青光眼"。编码员在病程中查到房角镜报告："双眼房角开放，宽角，色素分级1级"。OCT 显示双眼 RNFL 变薄，视野检查有弓形暗点。

问题：正确的诊断为？应编 H40.0 还是 H40.1？

点击查看答案

✅ H40.1 原发性开角型青光眼，排除 H40.0

推理过程：

① 房角镜："房角开放，宽角" → 开角型，排除闭角型 H40.2

② OCT："RNFL变薄" + 视野"弓形暗点" → 已有视神经损害，排除 H40.0（可疑青光眼）

③ 无继发因素（外伤/炎症/药物）→ 原发性

④ 最终：H40.1 原发性开角型青光眼

进阶题 · 糖尿病眼病星剑号综合编码 CHALLENGE

患者为2型糖尿病10年，本次因视力下降入院。眼科检查示：双眼糖尿病性视网膜病变（非增殖期）+ 右眼糖尿病性白内障。肾功能正常，无神经病变。

问题：请给出完整的编码方案，包括主诊断和所有附加码。

点击查看答案

✅ 完整编码方案：

主诊断：E11.3† 2型糖尿病伴有眼的并发症（剑号——病因）

附加1：H36.0* 糖尿病性视网膜病变（星号——眼底表现）

附加2：H28.0* 糖尿病性白内障（星号——晶状体表现）

推理过程：

① 仅有眼部并发症（视网膜+白内障），无肾/神经病变 → 不满足 E11.7 的"多系统"条件

② 因此用 E11.3†（仅涉及眼部），保留剑号以带动星号

③ E11.3† 可以同时带动 H36.0* 和 H28.0* 两个星号——这是允许的，属于同一系统的多个星号表现

💡 常见错误：试图用 E11.7（多并发症），导致丢失星剑号联动

进阶题 · 跨章边界：外伤性眼病 vs H章眼病 CHALLENGE

患者因"右眼钝挫伤后视力下降3天"就诊。CT示：右眼玻璃体积血、视网膜水肿。病程记录：3天前打篮球时被肘部撞击右眼。

问题：主诊断应编 H43.1（玻璃体出血）还是 S05.1（眼球挫伤）？给出完整编码。

点击查看答案

✅ 完整编码：

主诊断：S05.1 眼球和眶组织挫伤

附加1：W51.0（撞击损伤，体育运动中）——外因编码

❌ 不使用 H43.1（玻璃体出血）

推理过程：

① 病因明确为外伤 → 外伤性疾病归入 S 章（损伤），不在 H 章

② S05.x 覆盖了眼球挫伤及其各种表现（包括玻璃体出血、视网膜水肿）

③ H43.1 编码的是"内源性/自发性玻璃体出血"，如视网膜裂孔、糖尿病等引起的

④ 需要外因编码 W51.x 说明受伤机制（体育活动）

💡 口诀："有外伤就进 S 章，H 章专收内源病。"

进阶题 · 听力损失三轴分类综合判断 CHALLENGE

听力中心报告："纯音测听：右耳气导65dB，骨导30dB，气骨导差35dB；左耳气导70dB，骨导65dB，气骨导差5dB。声导抗：右耳鼓室图B型，左耳正常型。"

问题：请分析左右耳分别是什么类型的听力损失，并给出对应编码。

点击查看答案

✅ 分析：

右耳：气骨导差35dB（＞15dB），鼓室图B型 → 传导性听力损失 → H90.1（单侧传导性聋，对侧听力无损害）

左耳：气骨导差5dB（≤15dB），骨导已降至65dB → 感音神经性听力损失 → H90.4（单侧感音神经性聋，对侧听力无损害）

汇总：右耳传导性聋（H90.1）+ 左耳感音神经性聋（H90.4），两个码都需要。虽然不是混合性聋（混合性指同一侧耳朵同时有传导性和感音神经性成分），但两侧不同类型需要分别编码。

💡 核心规则：

  • 传导性聋：气骨导差＞15dB，骨导正常或接近正常
  • 感音神经性聋：气骨导差≤15dB，气导和骨导同步下降
  • 混合性聋（H90.6-H90.8）：同一侧耳气骨导差＞15dB 且 骨导也明显下降
  • 编码前必须看纯音测听+声导抗报告，不能凭诊断名称判断

✅ 八、今日重点回顾

H章一分为二：H00-H59 眼、H60-H95 耳，两条总线独立运行但交叉时需跨章联动
白内障三维编码：病因（六轴分类）+ 部位（核/皮质/囊膜）+ 形态（初期/成熟/过熟），缺一不可
青光眼核心轴心：房角状态决定一切——看房角镜报告，不要猜
中耳炎四象限：急性~慢性 × 化脓性~非化脓性，慢性化脓性再按穿孔位置分 H66.1/H66.2
听力损失看报告：气骨导差＞15dB→传导性；气骨导同步下降→感音神经性；两侧不同需分别编码
外伤不进H章：明确外伤导致的眼病→S 章（损伤）+外因码，不是 H 章
先天不进H章：先天性眼病/耳病→Q 章（先天性畸形），不是 H 章
H40.0 是红线：可疑青光眼≠确诊青光眼，有 OCT+视野损害证据即可排除 H40.0
